# Unit 6: Overview

Introduction

Now that we’ve learned about data representation, circuits, and memory hierarchy, we’re ready to look at computer architecture. We will start from the computer processor architecture and study von Neumann architecture, an architecture model that all contemporary processors adhere to. We will use Little Man Computer, a software simulator to understand how processors operate.
