Course Details

Title: Computer Architecture

Course Description: CS319 Computer Architecture: The student will learn about the various hardware components of a computer system. Course topics include: data representation, number systems, Boolean algebra, combinational logic, sequential logic, CPU layout, registers, adders, buses, and memory devices.

Credit Hours: 3:0:3

Prerequisites Courses: CS208 and either CS151 or CS152

Core Learning Outcomes

At the conclusion of the course, learners will be able to:

Solve problems involving Boolean algebra, integer computer representation and arithmetic (such as binary, octal, hexadecimal, 2’s complement), and floating-point computer representation and arithmetic.

Design combinational and sequential circuits. Explain various circuitry components including gates, multiplexers/demultiplexers, encoders/decoders, latches, flip-flops, registers, counters, and adders.

Explain and analyze Memory and I/O designs.

Explain and analyze CPU designs.

Course Assignments.

Discussion Questions (150 points) - There will be discussion questions in Units 1- 8.  Initial posts are due by 11:59 p.m. Wednesday, CT (Central Time).  After posting your answers, begin making 2 or more replies to other students' posts. 

Homework Assignments (485 points) – Units 1-7 contain homework assignments that are due by 11:59 p.m. Sunday, CT (Central Time).   These assignments consist of questions, critical thinking problems, and research papers.

Digital Logic Projects (155 points) – Units 2 and 4 have projects requiring the student to use digital logic simulation software to create and test digital logic circuits.   The projects are due by 11:59 p.m. Sunday, CT (Central Time).

Final Exam (210 points) – Unit 8 contains an online final exam that must be completed by 11:59 p.m. Sunday, CT (Central Time).

The final exam is closed book but you may have a single page of notes and a calculator:

one page note sheet: letter size (8.5 x 11”), hand-written, double-sided

the built-in calculator on your computer, but only the standard/basic mode is allowed. No scientific/programmer/advanced modes. Mac users should only use the calculator app at /Applications/Calculator.

Use of the Internet is not allowed during the exam
