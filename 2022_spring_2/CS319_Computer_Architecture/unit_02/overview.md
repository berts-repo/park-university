# Unit 2: Overview

Introduction

The last part of our study of data representation is floating-point numbers. We will learn how floating-point numbers are represented in IEEE 754 format. After we understand how IEEE 754 format works, we will investigate the limitation of storing real numbers in computers and a few key concepts such as rounding and overflow.

Having understood how data is represented in computers, now we're ready to learn how electronic circuits are built to manipulate data stored as 1s and 0s. We will start from combinational circuits, the type of circuits that just react to current inputs. We will have a quick review of relevant concepts covered in CS/MA208 Discrete Math (boolean algebra, logic gates, and combinational circuit) and then explain k-map based boolean expression simplification.
