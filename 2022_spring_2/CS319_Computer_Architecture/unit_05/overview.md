# Unit 5: Overview

Introduction

Having learned how sequential circuits operate, we will see how memory devices may be built using 1-bit latches. We will examine the memory hierarchy used by computers and learn about the different types of memory devices. We will see how adding cache into the memory hierarchy may improve the average memory access and how different cache mapping algorithms, direct mapping, fully associative mapping, and set-associative mapping work. 

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Compare different types of devices in the computer memory hierarchy and describe an example use of ROM, SRAM, and DRAM. (CLO 3)

Describe how a memory hierarchy with cache, main memory, and disk works. (CLO 3)

Solve addressing problems in a direct mapping/fully-associative/set-associative cache. (CLO 3)
