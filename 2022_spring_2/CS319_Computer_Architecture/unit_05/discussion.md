14. Given a computer with 34-bit addresses, Identify the number of bits used for tag number, set 
number, and word offset. The cache is an 8-way set-associative cache with 16K lines. The block size is 8.

n = 34

word offset = w: 

     block size = 8 = 2^w = 2^3

     w = 3

set number = s:

     k = 8

     16,000 / 8 = 2000

     2000 <= 2^s

     s = 11         

tag number = t:

     t = n - w - s

     t = 34 - 11 - 3 = 20
