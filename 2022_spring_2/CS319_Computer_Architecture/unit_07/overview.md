# Unit 7: Overview

Introduction

Last week we learned about von Neumann architecture, the architecture model behind modern computers. This week we will study an actual architecture, the x86 architecture. We will use an x86 assembly simulator to help us understand the x86 architecture and how the architecture determines higher software layers. 

Although the vast majority of data transactions within a computer occur between the processor and its memory, the processor must communicate with external devices to bring data into the computer (input) and to send out processing results (output). We will wrap up our discussion of architecture by understanding how computers perform input/output.
