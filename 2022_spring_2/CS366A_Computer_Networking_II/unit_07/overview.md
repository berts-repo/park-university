# Unit 7: Overview

Cisco CCNA Introduction to Networks v7 (ITN) Module 17

Unit Learning Outcomes

In this unit, we will learn how to build a small network.

ULO#1: Examine the devices, the protocols, and the applications that are used in a small network (CLO 1 & 2)

ULO#2: Carry out the network address calculation  (CLO 3)

ULO#3: Implement and secure a network with various network devices (CLO 4)
