# Unit 3: Overview

Cisco CCNA Introduction to Networks v7 (ITN) Module 12.8-12.9, and Module 13

Unit Learning Outcomes

In this unit, we will learn about implementing IPv6 addressing scheme, and testing the network connectivity.

ULO#1: Implement a subnetted IPv6 addressing scheme (CLO 3)

ULO#2: Use various tools to test network connectivity (CLO 4)
