# Unit 1: Overview

Cisco CCNA Introduction to Networks v7 (ITN) Module 11

Unit Learning Outcomes

In this unit, we will learn about the IPv4 Addressing.

ULO#1: Examine IPv4 addressing structure (CLO 3)

ULO#2: Calculate an IPv4 subnetting scheme to efficiently segment a network (CLO 3)
