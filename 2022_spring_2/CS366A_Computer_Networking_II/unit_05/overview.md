# Unit 5: Overview

Cisco CCNA Introduction to Networks v7 (ITN) Module 16

Note: Module 15 will be covered in the next unit.

Unit Learning Outcomes

In this unit, we will learn the network security fundamentals.

ULO#1: Examine the security vulnerabilities and the mitigation techniques  (CLO 4 & 5)

ULO#2: Configure switches and routers with device hardening features to enhance security (CLO 4 & 5)
