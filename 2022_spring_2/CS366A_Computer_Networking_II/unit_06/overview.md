# Unit 6: Overview

Cisco CCNA Introduction to Networks v7 (ITN) Module 15

Unit Learning Outcomes

In this unit, we will learn about the application layer.

ULO#1: Examine the application layer protocol (CLO 1 & 2)

ULO#2: Analyze the operation of application layer protocols in providing support to end-user applications (CLO 1 & 5)
