Hello class,

The protocol that interested me the most was Server Message Block (SMB), specifically SMB 3. The protocol allows applications to read and write to files to a computer over a network. SMB can be used on top of TCP/IP and other network protocols. Setting these folders up is relatively easy and it just takes just a few clicks.

There were two applications that caught my attention that use SMB, Hyper-V and Samba. Samba allows Linux and Unix machines to communicate with window machines, and Hyper-V allows for the storage and access to virtual machine files. Some more uses include Microsoft SQL Server, which can store user database files on SMB file shares, and traditional storage for end-user data.
