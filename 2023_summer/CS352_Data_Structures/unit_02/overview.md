# Unit 2: Overview

Introduction

This unit is an introduction to linked data structures, including linked lists. The discussion challenges you find real-life scenarios where a linked list is a practical data structure. The project lets you revisit your program from the previous unit, modify it to use a linked structure, gather timing information, then report on the performance differences between the two data structures.

Unit Learning Outcomes

ULO #1: Implement a Java program using a linked data structure (CLO 1 and CLO 2)

ULO #2: Apply algorithm analysis techniques to linear data structures (CLO 3)
