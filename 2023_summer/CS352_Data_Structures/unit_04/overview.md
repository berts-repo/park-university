# Unit 4: Overview

Introduction

This unit is an introduction to recursion and sorting algorithms that use recursion, such as merge sort and quicksort. Recursion is a technique where your program breaks a large problem into smaller pieces, and then solves each smaller piece using the same process. An example of this is merge sort, which breaks a list into two halves, sorts each half, then uses a merge process to create a single sorted list.
