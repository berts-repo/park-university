# Unit 6: Overview

Introduction

Now that you have practiced with linked data structures, we can examine a hierarchical structure, a tree, which allows for more flexibility than an array or linear structure. One example is that the files on your computer are organized as a tree, with folders and files within other folders. Trees are also often used when there are a lot of retrievals (or searching).

Unit Learning Outcomes

ULO #1: Implement a Java program using a binary tree (CLO 2 and CLO 4)

ULO #2: Apply algorithm analysis techniques to hierarchical data structures (CLO 3)

ULO #3: Critique the performance characteristics of binary trees (CLO 5)
