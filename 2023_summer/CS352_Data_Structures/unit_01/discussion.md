# Unit 1: Discussion

Introduction

This discussion will let you see appropriate and inappropriate uses of array-based lists when solving problems

Directions

Initial post

Propose a specific problem where an array-based list of objects is an appropriate data structure. Use objects besides Strings or Java primitive data types (int, double, etc.)

Describe the object including some attributes and any methods besides accessors/mutators.

Briefly describe how you add to the list, delete from the list, and how to search the list.

Example Initial Post

a) Maintain a list of phone contacts in alphabetical order by name

b) Each contact has a name and several phone numbers (work, cell, home). There are no methods besides accessors/mutators.

c) Adding and deleting are done infrequently, but must maintain the sorted order. Searching by name is done frequently

Peer Response

Respond to at least two student's proposals by commenting on the appropriateness of the data structure (either appropriate or inappropriate), adding additional components to the object, or proposing additional functionality to the list.

Example Peer Response

Because the list is sorted, lookups by name can be quick by using binary search. Looking up a contact by phone number would use sequential search, and there would have to be a way to differentiate all the different types of phone numbers
