# Unit 1: Overview

Introduction

This unit is an introduction to the interactive text, which includes a brief review of the List interface and of array-based lists. These were initially covered in CS 252. The discussion challenges you find real-life scenarios where an array-based list is a practical data structure. The project introduces you to using a version control system for saving and modifying your code.

Unit Learning Outcomes

ULO #1: Implement a Java program using an array-based list (CLO 1)

ULO #2: Summarize object-oriented concepts (CLO 2))

ULO #3: Critique the performance characteristics of an array-based list (CLO 5)
