# Unit 5: Overview

Introduction

While a previous unit introduced you to linked structures in general, this unit focuses on lists in particular. There are several variations, including lists that facilitate moving both forward and backward, as well as conveniently traversing one end to wrap around to the other end (a circular list). The programming assignments focuses on using linked lists in various ways, and then comparing the performance of the different approaches.

Unit Learning Outcomes

ULO #1: Implement a Java program using a linked list (CLO 2)

ULO #2: Critique the efficiency of various sorting algorithms (CLO 3)

ULO #3: Apply algorithm analysis techniques to linear data structures (CLO 3)
