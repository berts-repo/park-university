I am sharing  [link: https://en.wikipedia.org/wiki/Krauss_wildcard-matching_algorithm] Krauss wildcard-matching algorithm, this is a pattern matching algorithm that implements a couple different basic wildcard functionality. This algorithm is non-recursive and allows the user to implement a more intuitive syntax rather than regular expression. This algorithm is open-source and has been tested for reliability and peformance. The algorithm only uses two while loops and has multiple implementations including a portable version that uses indexes instead of pointers. (Krauss, 2023)

The algorithm works with the characters * and ?

A one-to-one match is performed between the pattern and the source to be checked for a match, with the exception of asterisk (*) or question mark (?) characters in the pattern.

An asterisk '*' character matches any sequence of zero or more characters.

A question mark '?' character matches any single character.

 

Krauss, K. J. (n.d.). Kirkjkrauss/MatchingWildcards: Matching Wildcards: An improved algorithm for Big Data. GitHub. https://github.com/kirkjkrauss/MatchingWildcards
