Course Details

Title: CS352 Data Structures

Course Description: This course introduces various data structures and advanced algorithmic concepts, using Java for the programming projects. These data structures will be covered: linked lists, stacks, queues, trees and hash tables. These concepts will be covered: recursion, sorting, and time-complexity analysis.

Credit Hours: 3:0:3

Prerequisite Courses: CS 208 and a grade of C or better in CS 252

Core Learning Outcomes

At the conclusion of the course, learners will be able to:

Given a problem description, construct a Java program that uses algorithms and data structures from the Java Collections Framework.

Given a problem description, construct a Java program that implements an abstract data type using generics and linked objects.

Analyze the efficiency of algorithms using Big-O notation.

Implement recursive methods and distinguish appropriate from inappropriate uses of recursion.

Distinguish the characteristics of data structures such as linked lists, stacks, queues, priority queues, trees, heaps, dictionaries and hash tables.
