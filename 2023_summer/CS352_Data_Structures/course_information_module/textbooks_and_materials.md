Software

You need to install a Java Development Kit/JDK first (if you don't have one) and then an appropriate IDE to develop Java programs on your own computer. 

 [link: https://parkuniversity.freshdesk.com/support/solutions/articles/6000177801-how-do-i-access-the-computer-lab-applications-from-a-personal-computer-device-]  You may use Park's Virtual Lab    (Note: you should use "CS_Lab" under Desktop tab, not "Student Lab". "CS Lab" is an image specifically for CS courses) before getting your computer ready. The virtual desktop has JDK, JavaFX, and Eclipse (IDE) already.

An online Java compiler (with IDE interface) such as   [link: https://www.onlinegdb.com/online_java_compiler]  Online Java Compiler  may be helpful when you need to test snippets of code or when your own IDE/JDK is temporarily unavailable.  

Installation

Java Development Kit (JDK): If you need to install a JDK, the latest long-term support (LTS) version (currently LTS 17)- will save you hassles . Get OpenJDK from   [link: https://adoptium.net/]  Eclipse Adoptium/AdoptOpenJDK .

See online resources such as  [link: https://docs.google.com/document/d/1nFGazvrCvHMZJgFstlbzoHjpAVwv5DEdnaBr_5pKuHo/]  Java Is Still Free   for an explanation of JDK vs. OpenJDK and versions.

An Integrated Development Environment (IDE): ideally you should use an industry IDE such as  [link: http://www.eclipse.org/downloads/]  Eclipse   or  [link: https://code.visualstudio.com/docs/languages/java]  Visual Studio Code    (VS Code). Other possible IDEs such as  [link: https://netbeans.org/downloads/]  NetBeans  work too.
