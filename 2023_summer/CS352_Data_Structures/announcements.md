## Welcome to Data Structures
This course is about using and choosing data structures that generally make solving a particular problem faster. 

It's a unique course in that the text is an online set of readings, animations, and user-interactions. It's built into the Canvas shell, and there is no charge.

You will be writing Java programs this term, so you should either make sure you have a Java compiler set up on your own computer, or learn to use Park's virtual desktop, CS Lab. Instructions are in the Course Information Module.

You will also need a free account on GitHub. If you have one already, you're all set. Otherwise, go to and  [link: https://github.com/] https://github.com/ and create an account. We'll be using this in Unit 1, so get that done as soon as you can. 

The final exam will be online, but proctored. You need a computer with strong internet connection, a web cam, and a microphone. More details are in the course modules.

There is a lot of programming in this class, and you may want to consider taking it as your only class. But it is rewarding as you see ways of improving your code and solving problems that you might not have thought possible before.

Let me know if you have any questions.
