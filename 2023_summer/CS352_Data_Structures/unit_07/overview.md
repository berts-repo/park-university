# Unit 7: Overview

Introduction

Trees are an efficient way to store data that allow for faster searching than linear structures. But there are ways of storing data that allow for even faster lookup. If you think about arrays, it takes the same amount of time to access the first element as the last. In this unit you’ll discover hash tables which allow similar fast access to item, but instead of looking them up by position, 0,1,… 100 etc. we’ll look them up by a key.

Unit Learning Outcomes

ULO #1: Implement a Java program using a hashed data structure (CLO 1)

ULO #2: Apply algorithm analysis techniques to hierarchical data structures (CLO 3)
