# Unit 1: Overview

Cisco CCNA – Switching, Routing, and Wireless Essentials v7 Module 1 and Module 2

Unit Learning Outcomes

In this unit, we will learn about the basic device configuration, and the various switching concepts.

ULO#1: Configure devices using security best practices (CLO 3 & 4)

ULO#2: Explore how layer-2 switches forward data (CLO 4)
