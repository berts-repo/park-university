# Unit 3: Overview

Cisco CCNA - Switching, Routing, and Wireless Essentials v7 Module 6, Module 7, and Module 8

Unit Learning Outcomes

In this unit, we will learn about EtherChannel, DHCPv4, DHCPv6, and SLAAC.

ULO#1: Troubleshoot EtherChannel on switched links (CLO 3)

ULO#2: Implement DHCPv4 to operate across multiple LANs (CLO 1, 3 & 4)

ULO#3: Configure dynamic address allocation in IPv6 networks (CLO 3 & 4)
