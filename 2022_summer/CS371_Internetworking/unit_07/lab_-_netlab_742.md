> Bert,
Well done on your assignment.  The DHCP server plays a major role on most networks.  It is used on small home networks up to the enterprise wide networks.  The IP addresses that are acquired from a DHCP server have a lease time associated with them.  This is specially useful when, for some reason, there is a limited number of IP addresses to be used.  When the lease expires the IP address can be recycled and used by other workstations.  We do have to remember that not every network device can use a dynamic IP address.  For example, routers or servers should not use a dynamic IP address since it may change.  For those devices we need to assign a static IP address manually.
