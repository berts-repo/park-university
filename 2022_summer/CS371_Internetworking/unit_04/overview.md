# Unit 4: Overview

Cisco CCNA - Switching, Routing, and Wireless Essentials v7 Module 9, Module 10, and Module 11

Unit Learning Outcomes

In this unit, we will learn about FHRP concepts, LAN security concepts, and switch security configuration.

ULO#1: Explore how FHRPs provide default gateway services in a redundant network (CLO 1 & 2)

ULO#2: Examine how vulnerabilities compromise LAN security (CLO 4)

ULO#3: Configure switch security to mitigate LAN attacks (CLO 3 & 4)
