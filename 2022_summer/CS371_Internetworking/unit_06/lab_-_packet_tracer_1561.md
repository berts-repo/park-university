Gateway of the last resort will provide the default route for your router which is very useful.  Lets keep in mind that the router will go through all the other entries in the route table first and if there is a match to the destination then it will use that route, however if there is no match found then it will use the gateway of the last resort.  If there is no gateway of the last resort then the packets will get dropped.
