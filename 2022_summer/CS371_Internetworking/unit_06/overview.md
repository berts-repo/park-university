# Unit 6: Overview

Cisco CCNA - Switching, Routing, and Wireless Essentials v7 Module 14, Module 15, Module 16

Unit Learning Outcomes

In this unit, we will learn about routing concepts, IP static routing, and troubleshoot static and default routes.

ULO#1: Explore how routers use information in packets to make forwarding decisions (CLO 1 & 2)

ULO#2: Configure IPv4 and IPv6 static routes (CLO 3)

ULO#3: Troubleshoot static and default route configurations (CLO 3 & 4)
