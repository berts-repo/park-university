# Unit 2: Overview

Cisco CCNA – Switching, Routing, and Wireless Essentials v7 Module 3, Module 4, Module 5

Unit Learning Outcomes

In this unit, we will learn about VLANs, inter-VLAN routing, and STP Concepts.

ULO#1: Implement VLANs and trunking in a switched network (CLO 1, 3 & 4)

ULO#2: Troubleshoot inter-VLAN routing on Layer 3 devices (CLO 1 & 3)

ULO#3: Explain how STP enables redundancy in a Layer 2 network (CLO 1 & 2)
