# Unit 5: Overview

Cisco CCNA -Switching, Routing, and Wireless Essentials v7 Module 12 and Module 13

Unit Learning Outcomes

In this unit, we will learn about WLAN concepts and configuration.

ULO#1: Explore how WLANs enable network connectivity (CLO 1)

ULO#2: Implement a WLAN using a wireless router and WLC (CLO 1, 3 & 4)
