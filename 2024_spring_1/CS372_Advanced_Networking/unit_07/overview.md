# Unit 7: Overview

Cisco CCNA – Enterprise Networking, Security, and Automation v7 Module 13 and Module 14

Unit Learning Outcomes

In this unit, we will learn about the network virtualization, and network automation.

ULO#1: Examine the purpose and characteristics of network virtualization. (CLO 3 & 4)

ULO#2: Explore how network automation is enabled through RESTful APIs and configuration management tools. (CLO 3 & 4)
