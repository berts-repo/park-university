# Unit 4: Overview

Cisco CCNA - Enterprise Networking, Security, and Automation v7 Module 7 and Module 8

Unit Learning Outcomes

In this unit, we will learn about the WAN, VPN, and IPsec concepts.

ULO#1: Examine how WAN access technologies can be used to satisfy business requirements. (CLO 1)

ULO#2: Explore how VPNs and IPsec are used to secure site-to-site and remote access connectivity. (CLO 1 & 2)
