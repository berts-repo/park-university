# Unit 5: Overview

Cisco CCNA - Enterprise Networking, Security, and Automation v7 Module 9 and Module 10

Unit Learning Outcomes

In this unit, we will learn about the QoS concepts and network management.

ULO#1: Examine how networking devices implement QoS. (CLO 3 & 4)

ULO#2: Implement protocols to manage the network. (CLO 3)
