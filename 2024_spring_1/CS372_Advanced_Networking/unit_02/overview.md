# Unit 2: Overview

Cisco CCNA –Enterprise Networking, Security, and Automation v7 Module 3 and Module 4

Unit Learning Outcomes

In this unit, we will learn about the network security concepts and ACL concepts

ULO#1: Examine how vulnerabilities, threats, and exploits can be mitigated to enhance network security. (CLO 2 & 4)

ULO#2: Explore how ACLs are used as part of a network security policy. (CLO 2 & 4)
