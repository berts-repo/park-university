Netlab 3.8.8

Netlab 3.8.8 is a Wireshark lab. It requires Wireshark software and connectivity to the Internet. In FHSU Netlab environment, Wireshark is installed. However, FHSU Netlab is a closed environment, meaning that it does not allow users to explore traffic outside of Netlab environment. Therefore, you will  [link: http://www.wireshark.org/download.html] install Wireshark in your local device. If you cannot install Wireshark in your computer, you can use the Wireshark which has already installed in the Cisco lab within Park Citrix Virtual Desktop (citrix.park.edu.) Detailed information about Wireshark, requirements, and how-to can be found under “ [link: https://canvas.park.edu/courses/78753/pages/faq-packet-tracer-netlab-wireshark] FAQ: Wireshark.”
