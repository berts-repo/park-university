# Unit 3: Overview

Cisco CCNA - Enterprise Networking, Security, and Automation v7 Module 5 and Module 6

Unit Learning Outcomes

In this unit, we will learn about the ACLs for IPv4 configuration and NAT for IPv4

ULO#1: Implement IPv4 ACLs to filter traffic and secure administrative access. (CLO 2)

ULO#2: Configure NAT services on the edge router to provide IPv4 address scalability. (CLO 1 & 2)
