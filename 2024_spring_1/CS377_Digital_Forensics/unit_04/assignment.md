Entry 0-10 are direct disk block pointers;

Entry 11 is a single indirect pointer

Entry 12 is a double indirect pointer

Entry 13 is a triple indirect pointer.

2) Compute the size range of files that use both direct disk block pointer and single indirect pointer.

3) Computer the size range of files that use direct block pointer, single indirect pointer, and double indirect pointer.

4) Compute the maximal file size this system can manage.
