# Unit 3: Overview

Unit 3: Windows Forensics and Forensics Tools

Introduction 

Understanding technical details is essential when performing forensics for certain OS. In this unit, you will review disk structure, file system, registry, and startup tasks of Windows OSs. In addition, you will also explore different software and hardware tools used during digital forensics.

 Click for Unit Learning Outcomes

ULO1: Explain how to prepare a digital forensics investigation by taking a systematic approach.

ULO2: Describe standard procedures for conducting forensic analysis.

ULO3: Apply different computer forensic tools to a given cybercrime scene.

UL43: Apply current practices to data recovery and acquisition.
