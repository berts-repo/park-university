# How This Course Works- Please Read

This course is designed using Cengage Mindtap. Follow the steps below to access the course content within Cengage Mindtap. The Cengage Mindtap assignments are deep linked within the course. If you have any issues please navigate to  [link: https://startstrong.cengage.com/mindtap-canvas-ia-no/] Cengage Technical Support. 

Click on Welcome to Your Course in the Course Information Module OR any assignment within Unit 1 and navigate to the Live Virtual Lab link to get started with Cengage Mindtap.  

Navigate to the Cengage homepage where you will authorize access to the content for this course. 

Click to Authorize Cengage Learning using the access code you purchased from  [link: https://bncvirtual.com/park] MBS Bookstore.  (Please note: You will only need to authorize access one time.) 

Once the content has been authorized, you will use the assignment links within the Canvas module to access all assignments and quizzes in Cengage Mindtap.
