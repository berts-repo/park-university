# Final Exam Study Guide

Final Exam

Access the exam via Honorlock on the global navigation menu at left.

This exam consists of approximately 25 multiple choice, multiple response, true/false, or matching questions, 3 short essay questions, and 2 calculation questions.

Each short essay or calculation question is 10 points. Each other type of question is 2 points.

During the exam, there will be one question displayed at a time. However, you can change the answer of each question at any time before submitting the exam. 

It has a time limit of 120 minutes.

It can be taken only once.

It is proctored via Honorlock.

The exam is close book, close note. However, one double-sided cheat sheet is allowed. The cheat sheet has to be hand written and in the US-letter size.

Scratch papers are allowed. You will be required to show your scratch paper to the proctor and destroy it on camera before you will be permitted to exit the testing session.

An alternative to scratch paper is a white board, or a transparent sleeve with a blank sheet of paper in it that can be written on with a dry-erase marker, and then erased at the end of the exam.

Before taking this exam, it is highly recommended to review all previous quizzes, labs, and assignments. 

Topics

Unit 1:

At the completion of this unit, the learner will be able to:

Describe the field of digital forensics (CLO 1)

Describe how to prepare a digital forensics investigation by taking a systematic approach (CLO 1)

Summarize how to conduct an investigation, including critiquing a case (CLO 2)

List physical requirements for a digital forensics lab (CLO 1)

Describe components used to build a business case for developing a forensics lab (CLO 1 and 3)

 

Unit 2:

At the completion of this unit, the learner will be able to:

List digital evidence storage formats (CLO 1 and CLO 3)

Explain ways to determine the best acquisition method (CLO 4)

Describe contingency planning for data acquisitions (CLO 4)

Explain how to use acquisition tools and how to validate data acquisitions (CLO 3 and CLO 4)

Explain the rules for controlling digital evidence and processing law enforcement crime scenes (CLO 2)

Explain procedures for storing digital evidence and obtain a digital hash (CLO 2)

 

Unit 3:

At the completion of this unit, the learner will be able to:

Describe Microsoft file structures and the structure of NTFS disks (CLO 3)

List some options for decrypting drives encrypted with whole disk encryption (CLO 3)

Explain the Windows Registry and Microsoft startup tasks (CLO 3)

Describe available digital forensics software tools (CLO 3)

List some considerations for digital forensics hardware tools (CLO 3)

Describe methods for validating and testing forensics tools (CLO 3)

 

Unit 4: 

At the completion of this unit, the learner will be able to:

Describe Linux and Macintosh file structures (CLO 3)

Use Linux forensics tools (CLO 3)

Describe types of graphics file formats and data compression (CLO 4)

Explain how to locate and recover graphics files (CLO 4)

Describe how to identify unknown file formats (CLO 4)

Explain copyright issues with graphics (CLO 4)

 

Unit 5:

At the completion of this unit, the learner will be able to:

Determine what data to analyze in a digital forensics investigation (CLO 2)

Explain tools used to validate data (CLO 3)

Explain common data-hiding techniques (CLO 4)

Explain standard procedures for conducting forensic analysis of virtual machines (CLO 2)

Describe the process of a live acquisition (CLO 2)

Describe standard procedures in network forensics and network-monitoring tools(CLO 2)

 

Unit 6:

At the completion of this unit, the learner will be able to:

Describe tasks in investigating e-mail crimes and violations (CLO 2)

Describe some specialized e-mail forensics tools (CLO 3)

Describe procedures for acquiring data from mobile devices (CLO 2)

Explain the challenges of forensic acquisitions of data stored on Internet of Anything devices (CLO 2)

 

Unit 7:

At the completion of this unit, the learner will be able to:

Explain the legal challenges and technical challenges in conducting cloud forensics (CLO 2)

Describe how to acquire cloud data and conduct cloud investigation (CLO 2)

Explain what remote access tools can be used for cloud investigations (CLO 3)

Describe guidelines for writing reports (CLO 2)

Explain how to use forensics tools to generate reports (CLO 3)

 

 Unit 8: 

At the completion of this unit, the learner will be able to:

Explain guidelines for giving testimony as a fact witness or an expert witness (CLO 2)

Describe guidelines for testifying in court (CLO 2)

Explain guidelines for testifying in depositions and hearings (CLO 2)

Describe procedures for preparing forensics evidence for testimony (CLO 2)

Explain how ethics and codes apply to expert witnesses (CLO 2)

Explain the process of carving data manually (CLO 3)
