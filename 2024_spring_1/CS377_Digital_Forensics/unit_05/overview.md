# Unit 5: Overview

Unit 5: Forensics Validation, Virtual Machine and Network Forensics

Introduction 

In this chapter, you learn more about using hashing algorithms in forensics analysis to validate data. You also learn how to refine and modify an investigation plan, use data analysis tools and practices to process digital evidence, determine whether data-hiding techniques have been used, and learn methods for performing a remote acquisition. Additionally, you will also explore procedures for virtual machine forensics, live acquisitions, and network forensics.

 Click for Unit Learning Outcomes

ULO1: Explain how to prepare a digital forensics investigation by taking a systematic approach.

ULO2: Describe standard procedures for conducting forensic analysis.

ULO3: Apply different computer forensic tools to a given cybercrime scene.

UL43: Apply current practices to data recovery and acquisition.
