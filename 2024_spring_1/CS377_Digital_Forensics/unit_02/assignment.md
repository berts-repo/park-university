Answer the following questions with detailed commands. 

You are asked to acquire the disk /dev/sdb with command dd and save the acquired image file into partition /dev/sda1. For example, if the image has been split into three files, the names of the split files should be sdb.aa, sdb.ab, and sdb.ac. a) Describe the steps and each commands, especially the dd command, that are required to complete the task; and 2) Based on the dd command output, compute how many files will be generated.

Describe steps and commands that can be used to validate the acquired data.
