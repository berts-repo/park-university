# Unit 3: Overview

Introduction

In order to adapt to more general problems, AI agents require a flexible approach to representing knowledge that extends beyond the python code you worked with in Unit 2. In this unit, you’ll work with your peers to utilize predicate logic for reasoning and representing knowledge. Combining this with your knowledge of search, you will also begin working with planning algorithms and again modify some python planning code to solve a real-world problem. 

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Model narrow domains of real-world knowledge using predicate logic statements (CLO 3, 4)

Compute the result of a query using deductive inference (CLO 3)

Distinguish a planning approach from basic search techniques and recognize problems best suited to it (CLO 1, 2)

Formulate a planning problem space to address a real-world problem (CLO 3, 4, 5)

Employ a planning algorithm to solve a real-world problem (CLO 2, 3, 5)
