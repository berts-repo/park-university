# Unit 1: Overview

Introduction

In this unit, you’ll learn about the field of artificial intelligence and connect your python programming skills to the practice of employing AI methods for problem-solving. You’ll discuss the nature of problem-solving in AI and the agent perspective with your peers and work through some python refresher problems as you also gain comfort with the programming tools we’ll be using in this class.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Identify aspects of the Python programming language conducive to implementing AI algorithms (CLO 5)

Characterize an intelligent agent (CLO 4)

Apply an intelligent agent architecture to a real-world problem (CLO 2, 4)

Summarize significant events through AI history leading to its modern applications (CLO 2)
