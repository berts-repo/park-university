# Unit 2: Discussion - Search Problem Formulation

Introduction

The purpose of this assignment is to better understand search as a problem solving strategy. You will work together with your classmates to develop several specifications of search problems following the guidance of the text.

Directions

Before beginning this assignment, complete the reading assignment at least through Ch. 3.2 of AIFCA and the Intelligent Agents lecture notes. 

Consider the search examples mentioned in the text and lecture. Create a post naming and very briefly describing search problem of your choosing, separate from these examples. Your problem can be a puzzle or an abstract representation of a more practical problem. Briefly describe the state space for the search problem. 

Once there are a number of initial posts available in the discussion, choose one that has not yet been responded to. If you don’t feel the example meets the definition of an search problem well, briefly outline some changes or assumptions you would consider for the example. Next consider the possible states reachable from the original author’s described state. Additionally, specify what a goal state for this problem would be.  

Before completing this step you should complete all chapter 3 reading. Once most of the initial posts have been responded to, choose a response to a different post to respond to. Devise and describe a heuristic for this search problem. Identify whether the heuristic satisfies the monotone restriction and comment on what other choices could be made to optimize solving the search problem.
