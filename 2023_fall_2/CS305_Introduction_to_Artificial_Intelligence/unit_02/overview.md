# Unit 2: Overview

Introduction

Search is the most fundamental concept in automated reasoning and connects your study of AI back to the knowledge you gained in data structures (CS352). Many problems in AI can be expressed as a “Constraint Satisfaction Problem” (CSP) that can be solved using search algorithms. In this unit you’ll collaborate with your peers to determine practical problem specifications amenable to search and specify a search strategy for solving them. in your homework assignment you’ll start with some existing python code for solving CSP’s and make some changes to solve a practical problem.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Formulate a search problem space to address a real-world problem (CLO 4, 5)

Design an appropriate heuristic to optimize a search process (CLO 5)

Assess the feasibility and optimality of a heuristic search algorithm in the context of a real-world problem (CLO 1, 5)

Employ a constraint-satisfaction algorithm to solve a real-world problem (CLO 5)
