# Unit 6: Overview

Introduction

The prior unit examined reasoning when uncertainty was involved; in this unit you will explore how the inclusion of multiple agents (both cooperative and adversarial) in the reasoning process brings other complications that can be overcome with clever algorithmic approaches. You’ll also examine a third perspective on machine learning that sits between the supervised and unsupervised approaches you studied in previous units: reinforcement (or semi-supervised) learning. You’ll discuss which of these three categories best fits a particular learning problem with your peers, and you’ll begin working on your course project: to create your own competitive game-playing agent by making some changes to some existing python code implementing the minimax adversarial search algorithm. 

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Formulate an advanced search problem space to make decisions in a game environment (CLO 4, 5)

Diagram the performance enhancement of alpha-beta pruning in the minimax algorithm (CLO 1)

Design a competitive heuristic to deploy with the minimax search algorithm (CLO 5)

Compare strategies for decision making within a group of intelligent agents (CLO 1)

Distinguish reinforcement learning from other categories of machine learning (CLO 1)

Judge whether a problem is best approached by supervised, unsupervised, or semi-supervised (reinforcement) learning algorithm (CLO 1)
