# Unit 5: Overview

Introduction

Supervised learning can work well under certain conditions, but when labeled data is not available unsupervised approaches may be more applicable. This unit will explore another side of machine learning, unsupervised learning, as well as the complications that can arise in learning and reasoning tasks when uncertainty is considered. You will discuss how to formalize problems fitting this new category of machine learning with your peers and use python to apply unsupervised learning techniques to real-world problems. 

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Explain how probability theory can be applied in the reasoning process (CLO 1, 3)

Employ a probabilistic classifier to solve a real-world problem (CLO 4, 5)

Formulate an unsupervised machine learning problem definition to address a real-world problem (CLO 4, 5)

Employ an unsupervised learning algorithm to solve a real-world problem (CLO 2, 5)

Analyze the performance of an unsupervised learning algorithm or model on a variety of performance metrics (CLO 1, 2, 5)
