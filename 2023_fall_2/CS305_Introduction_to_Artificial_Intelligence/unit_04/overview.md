# Unit 4: Overview

Introduction

Over the last decade, Machine Learning has arguably become the most studied subfield within AI. This unit will help you understand the fundamental concepts behind supervised machine learning and several supervised learning algorithms. You will discuss how to formalize a machine learning problem with your peers and explore common algorithms implemented through AIPython tools for developing and evaluating supervised learning models. 

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Formulate a supervised machine learning problem definition to address a real-world problem (CLO 4, 5)

Classify problems solvable by ML techniques by attributes that dictate selection of appropriate ML algorithms and algorithm parameters (CLO 1, 4, 5)

Analyze the performance of a supervised learning algorithm or model on a variety of performance metrics and recognize the presence of over-fitting (CLO 1, 2, 5)

Characterize the structure of a neural network model and how that structure is enriched in deep learning (CLO 1, 5)

Distinguish lazy-learning techniques from other (eager) methods (CLO 1)

Explain how machine learning techniques can be integrated together to develop composite models or integrated with other AI techniques (CLO 1)
