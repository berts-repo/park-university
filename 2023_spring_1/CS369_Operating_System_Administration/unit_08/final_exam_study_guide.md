# Final Exam Study Guide

This exam is closed book and closed Internet. However, you may bring one small sheet of handwritten notes, (both sides) on a 5x7" index card or half sheet (5.5 x 8.5") of standard letter sized paper. 

The exam is entirely short answer. Questions will be displayed one at a time, but you will be able to move back and forth and may answer them in any order.

You have 2 hours and may only take the exam once.

The following topics may be covered on the exam:

Linux commands and directory tree structure – review tutorials from Unit 1

Linux shell – wildcards, redirection, aliases

Create, configure, start and stop GCP instances

Install and configure nginx web server

Programs/scripts to read data and process data from a file.

Explain the parts of an nginx configuration file

Linux files associated with user accounts/logins

Public/private key pairs, use with ssh to login

Install and configure MySQL/MariaDB

Install and configure php for use with nginx

Install and configure WordPress

Create and use GCP snapshots

Write crontab entries to run scripts at specified time intervals

Be sure to review all the discussions and assignments as well.
