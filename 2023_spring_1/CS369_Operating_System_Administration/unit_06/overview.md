# Unit 6: Overview

Introduction

This unit introduces you to installing and configuring a suite of complementary programs. You’ll be building a LEMP server, Linux, (E)nginx, MySQL (MariaDB), and PHP, and then installing a web application that makes use of all these services.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Install a database server (CLO 3)

Configure a database server for performance and enhanced security (CLO 2)

Install a php server (CLO 3)

Configure php for enhanced security (CLO 2)

Install a PHP-based web application (CLO 3)

Configure the web application for enhanced security (CLO 2)
