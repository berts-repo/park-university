# Unit 4: Overview

Introduction

This unit introduces you to configuring the nginx web server to host multiple web sites on a single server. You’ll also review a little about ports and firewalls, which are covered initially in our networking courses, and Linux archive formats.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

    
Configure nginx web server (CLO 3)

    
Control the nginx web server from the command line (CLO 5)

    
Open paths through a firewall (CLO 1)
