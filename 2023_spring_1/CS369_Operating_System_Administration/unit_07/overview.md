# Unit 7: Overview

Introduction

This unit introduces you to installing and configuring a new disk and file system, along with scheduling processes to run at later times, often on a fixed schedule.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Add a new disk and file system to your instance (CLO 4)

Configure your instance to mount a disk at boot up (CLO 4)

Configure your instance to run processes on a schedule without human intervention (CLO 3)

Contrast the benefits and performance of different file system organizations (CLO 4)
