# Unit 2: Overview

Introduction

This unit introduces you to the Google Cloud Platform (GCP), creating a virtual machine instance, and running a web server on your own instance. In order to run GCP, you will need a regular Google account, in addition to your Park Gmail account.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

    
Use the Linux command line interpreter (CLO 5)

    
Navigate a directory hierarchy (CLO 5)

    
Change file ownership and permissions (CLO 2)

    
Search for and edit text information in files (CLO 5)

    
Create, run and stop a virtual machine instance (CLO 3)

    
Install and configure a web server (CLO 3)
