# Unit 3: Overview

Introduction

This unit introduces you to doing system maintenance tasks using Python programs.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Use the Linux command line interpreter (CLO 5)

Navigate a directory hierarchy (CLO 5)

Edit text information in files (CLO 5)

Run and stop a virtual machine instance via an API (CLO 5)

Discuss logging and the implications for system security (CLO 2)

Readings and Materials

Textbook Readings

Linux System Administration: Chapter 17, Logging. You will not need to configure anything for this chapter but you should be familiar with the concept of logging and what is in the director /var/log.
