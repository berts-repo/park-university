# Unit 1: Overview

Introduction

This unit introduces you to the Linux command line and basic directory layout in a Linux system. You will complete several tutorials in a virtual Linux environment (LinuxZoo.net) and submit your performance documentation for instructor verification.

You will need to register for a  [link: https://linuxzoo.net/] Linux Zoo Account and use the virtual machines there to complete Unit 1 - Assignment 1. Instructions are in the assignment. Get your account early and at least make sure you can log in and use the system early in the unit.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Use the Linux command line interpreter (CLO 5)

Navigate a directory hierarchy (CLO 5)

Change file ownership and permissions (CLO 2)

Search for and edit text information in files (CLO 5)

Discuss Linux configurations (CLO 3)
