Absolute vs. Relative paths

A path that starts with a / means start at the root of the entire file hierarchy (/home/jc, /bin/ls)

A path that starts with a name means start in the current directory (thisfile, thisFolder)

A path that starts with ./ means start in the current directory (./thisfile, ./thisFolder)

A path that start with ../ means start in the directory immediately above this one

Changing directories

cd by itself takes you to your home directory. This is often not what you want.

cd - takes you back to where you were.
