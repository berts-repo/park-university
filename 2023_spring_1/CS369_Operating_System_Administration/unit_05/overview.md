# Unit 5: Overview

Introduction

This unit introduces you to user management. You will examine the mechanisms that control user access both within a Linux environment and also through GCP, which automatically updates a running instance.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

    
Describe files associated with user logins (CLO 2)

    
Generate and use public/private key pairs (CLO 2)

    
Configure remote logins (CLO 1)
