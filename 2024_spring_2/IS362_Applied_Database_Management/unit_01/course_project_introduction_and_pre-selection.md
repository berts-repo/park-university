# Unit 1: Course Project Introduction and Pre-Selection

Objective

In this course, students are required to produce a class project that demonstrates her/his skills in the design, creation, manipulation, and use of a database. Each student will have a different project with similar characteristics. Students must design a database for the project they chose, create it in MariaDB, populate it with data, and write programs that will access the database. Guidelines of what to do at each stage of this process will be given during the semester at various project milestones.

Activities

During the first unit of classes (Unit1), students must select one among the titles for the projects listed below. The student who selects a topic is basically claiming it, and s/he will keep the project for the term of the course. There will be one project per student, and each of these projects will be different. First come, first served. By the same token, once a student claims a project, s/he cannot change her/his choice. So think carefully before selecting, but be quick to select. Students who do not claim a topic by Sunday midnight CST on Unit 1 will be assigned a topic by the Instructor.
