# Unit 7 : Final Exam Study Guide

Final Exam Study Guide

The Final Exam for this course comprises the MySQL commands that you have been using during the semester and the Java or Python programming you have learned to deal with databases. The exam describes one database design and asks 17 questions in which students must answer with MySQL commands and 1 question in which students must write a small Java or Python code. You do not have to learn all these commands by heart. You are provided with a PDF document that contains all the commands you may need in the exam. You must download this document from the link below and print it beforehand to bring it to the exam. All these questions are worth 100 points.

The final exam for this course has the following characteristics:

It is proctored.

It is a 2-hour final that can only be taken once. (This is school and department policy)

It is closed book.
No calculators are allowed.

Limited notes will be allowed. Students may use only the 4 pages of a List of Commands provided  [link: https://canvas.park.edu/courses/80245/files/11005431/download?verifier=n6Sg8zAN6TMaBlXdiiYuE4EdzWCxawyCaDijhsfB&wrap=1] here. Students should print this List beforehand to bring it to the exam.

Multiples sheets of scratch paper are allowed.

No use of additional software is allowed

Bathroom breaks are not allowed. Students should take care of their needs before starting the exam.

Only the Chrome Browser is allowed.

Each question in the exam will be presented to the student individually and students will not be able to go back to a question after it was answered and submitted.

Additional Examinations Notes:
Students may be flagged and warned about infractions while taking the exam, but they should be allowed to complete it within its allocated time period.
