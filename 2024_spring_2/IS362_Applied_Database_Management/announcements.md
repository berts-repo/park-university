## Final Exam
*Posted: 2024-04-27T14:56:45Z*

Hello Class,

In the event you didn't receive my email regarding Final Exam overview - see below.

 

I put together a study guide for the Final Exam, as you may already know you access the Final Exam through canvas Honorlock. The Final Exam is currently available and will remain open until May 5th around 10pm CST.

Be sure to take the time to study and prepare!

Here is what to expect on the Final Exam:

You will be provided a diagram of a Relational Data Model for a small company, it will contain 5 tables, show all the fields, identify all the PKs and also showcase their datatypes.

Following the RDM you will see a description of the company essentially describing each TABLE and the associated fields.

Based on the Database design you will be asked to answer questions by writing SQL commands.

Example, know the following clauses:

CREATE Table
ALTER TABLE and ADD CONSTRAINTS
INSERT INTO command to add records
MODIFY COLUMN and UPDATE
Drop records – DELETE FROM

Be able to formulate an SQL query based on a request; example display all IDs with License plate number of ?.

Write a query that will display weight. (SELECT SUM)

Display the count ( COUNT)

Display data between certain dates (>= and <=)

Display a particular number with weight above the average (Subquery) SELECT AVG

Also be familiar with DAYNAME, CREATE USER, GRANT clause

Be able to: Write a mysqldump command to backup all tables and command to restore all tables.

Lastly you will be given a SQL command such as a ‘create table command’ and be asked to put it in a string and write the statements needed to execute the command from Java or Py. You won’t have to write the whole program, just the minimum commands that are relevant for the task.

Hope this study guide helps, best of luck on the Exam.

## List of MySQL commands 
*Posted: 2024-03-22T14:49:02Z*

Hello CLass, below is a List of MySQL commands that you may find to be hepful.

Note that all text commands must be first on line and end with ';'
?               (\?) Synonym for `help'.
clear         (\c) Clear the current input statement.
connect   (\r) Reconnect to the server. Optional arguments are db and host.
delimiter  (\d) Set statement delimiter.
ego          (\G) Send command to mysql server, display result vertically.
exit          (\q) Exit mysql. Same as quit.
go            (\g) Send command to mysql server.
help         (\h) Display this help.
notee       (\t) Don't write into outfile.
print         (\p) Print current command.
prompt    (\R) Change your mysql prompt.
quit          (\q) Quit mysql.
rehash     (\#) Rebuild completion hash.
source     (\.) Execute an SQL script file. Takes a file name as an argument.
status      (\s) Get status information from the server.
tee          (\T) Set outfile [to_outfile]. Append everything into given outfile.
use         (\u) Use another database. Takes database name as argument.
charset   (\C) Switch to another charset. Might be needed for processing binlog with multi-byte charsets.
warnings  (\W) Show warnings after every statement.
nowarning (\w) Don't show warnings after every statement.
