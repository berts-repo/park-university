Required and Optional Materials:

Software: Python compiler

Initially, all programming can be done online, either through the textbook or with an online system like ideone.com. 

However, starting in Unit 6, you will be writing programs that manipulate text files. Unfortunately, there's no way to do this using the textbook. So, you'll have to have access to a standalone Python compiler.

The easiest way if you control your own computer is to just install the compiler from  [link: http://thonny.org] http://thonny.org.

If you don't want to install this compiler on your own computer, or you are running on a Chromebook, you can also use Park's virtual desktop. Here are a few videos to help you get started with this:
