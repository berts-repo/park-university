Course Details

Title:  CS152 Introduction to Python

Course Description: This course introduces students to computer programming and algorithmic thinking. The current language is Python. Students will learn about variables, data types, input/output, control flow, sequences, and functions.

Credit Hours: 3:0:3

Prerequisite Courses: A grade of "C" or better in any math course >= MA125, or grade of "C" or better in CS144, CS145, or CS147, or an ACT math score >= 23, or an SAT math score >= 510, or a College Success Algebra test score >= 110.

Core Learning Outcomes

At the conclusion of the course, learners will be able to:

Implement a Python program that uses iteration, branching and function calling based on a written problem description.

Analyze a Python program to determine syntax, runtime, and logic errors.

Implement Python functions with parameters.

Use collections such as lists, strings, tuples and dictionaries in a Python program.

Access text files for input and output.
