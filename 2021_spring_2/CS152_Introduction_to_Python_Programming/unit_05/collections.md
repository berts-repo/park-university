# Unit 5: Collections

Introduction

Unit Introduction: In this unit you will learn about using two different collections of data, strings, which are a specialized collection of characters, and lists, which are a collection of any kind of object. You’ll learn about the methods associated with each collection and ways of accessing and changing pieces of each collection. You’ll see that data in these collections are accessed by their position in the collection.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Explain how to access characters in a string by their position. (CLO4)

Apply the accumulator pattern to construct new strings.(CLO1 and CLO4)

Explain list slices. (CLO4)

Contrast strings and lists. (CLO4)

Implement a program using string methods and operators. (CLO1, CLO3, and CLO4)
