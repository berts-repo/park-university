# Unit 2: Starting to Think Like a Computer Scientist

Introduction

In this unit, you will learn more about different types of errors along with ways of finding them and preferably, preventing them in the first place. You will also learn about drawing images using turtle graphics and writing loops to perform repetitive actions.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Examine error messages to determine the location of a problem in a program. (CLO2)

Describe the different turtle drawing methods. (CLO1)

Use functions from the turtle module. (CLO1)

Implement a program that draws shapes using iteration. (CLO1)
