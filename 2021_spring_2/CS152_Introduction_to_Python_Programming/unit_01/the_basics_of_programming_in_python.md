# Unit 1: The Basics of Programming in Python

Introduction

In this unit you’ll learn a little about what it means to program a computer and what a programming language looks like. You’ll be introduced to the language Python, and components like variables, types, operators, assignment statements, and input/output. You’ll begin using the interactive textbook for readings and exercises and you’ll be writing simple programs before the end of the unit.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Explain the difference between syntax, runtime and logic errors (CLO2)

Execute and trace a python program (CLO2)

Explain the different basic data types (CLO1)

Implement a program using data types, user input, and simple operators (CLO1)
