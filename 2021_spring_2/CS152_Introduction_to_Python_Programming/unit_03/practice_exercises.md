# Unit 3: Practice Exercises

Introduction

If you want more practice coding functions, you can solve problems at  [link: https://www.codestepbystep.com] Code Step by Step website and these will be automatically tested and return you feedback about correctness.

Directions

For your own practice, go to  [link: https://www.codestepbystep.com] Code Step by Step website and create an account. Then choose Python as the language, and look in the section Parameters and return. You can try any of the problems in this section, but the following problems will be the most helpful to write new functions:

print_stars

average_of_3

box_of_stars

circle_area

compute_distance

get_last_digit

For a different type of problem involving practicing with parameters, see any of the ‘mystery’ problems like

mystery_xyz

parameter_mystery1

parameter_mystery2

parameter_mystery_exam1

etc
