# Unit 3: Procedural Programming

Introduction

Unit Introduction: In this unit you will about the concept of abstraction and encapsulation. You will first learn to call pre-defined functions to accomplish a task, then learn to create your own functions to encapsulate common operations. You will also learn about parameters and return values for communicating data to and from the functions you write.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Use functions from the math and random modules. (CLO1)

Explain the difference between fruitful and non-fruitful functions. (CLO3)

Recognize the accumulator pattern as another use for iteration. (CLO1)

Relate function parameters to loop variables. (CLO1 and CLO3)

Implement a program that contains fruitful and non-fruitful functions. (CLO1 and CLO3)
