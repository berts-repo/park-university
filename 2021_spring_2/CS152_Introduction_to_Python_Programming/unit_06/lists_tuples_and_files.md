# Unit 6: Lists, Tuples, and Files

Introduction

Unit Introduction: In this unit, you will continue studying lists, and learn about another collection, the tuple. You will also learn about reading data from a file and writing data to a file.

You must have access to a standalone python compiler to complete this unit's assignments. See  [link: https://canvas.park.edu/courses/59577/pages/textbooks-and-materials] Textbooks and Materials for how to get started.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Explain aliasing and cloning. (CLO4)

Contrast iteration patterns for processing vs. modifying a list. (CLO4)

Explain the concept of lists of lists. (CLO4)

Compare the different ways of reading from a file (CLO5)

Implement a program that reads data from a file and parses it into a list. (CLO1, CLO3, CLO4, and CLO5)
