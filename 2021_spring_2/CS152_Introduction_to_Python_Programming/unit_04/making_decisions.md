# Unit 4: Making Decisions

Introduction

Unit Introduction: In this unit you will learn about writing a program that behaves differently based on the data that it receives. You will also learn about Boolean values and expressions, as well as writing and using functions that return Boolean values. You will also learn about an alternative form of looping, which is suited to situations where the number of iterations depends on the data that it receives.   

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Explain the purpose of selection statements. (CLO1)

Explain the purpose of Boolean functions. (CLO1 and CLO3)

Compare a for loop with a while loop (CLO1)

Implement a program that uses Boolean functions, selection, and a while loop. (CLO1 and CLO3)

Implement a program using string methods and operators. (CLO1, CLO3, and CLO4)
