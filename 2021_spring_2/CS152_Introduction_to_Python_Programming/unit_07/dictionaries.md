# Unit 7: Dictionaries

Introduction

Unit Introduction: In this unit, you will learn about a different type of collection where the elements are not accessed by their positions, but by a key value instead. This is like looking up the definition of a word in the dictionary, or a phone number in a phone book.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Implement a program that modifies data in a file. (CLO1, CLO3, CLO4, and CLO5)

Describe dictionary methods. (CLO4)

Contrast lists and dictionaries. (CLO4)
