# Unit 1: Overview

Introduction

In this unit we will review the basic structures of C/C++ Programming. They will enable students to write, compile and run a program in Microsoft Visual Studio, capable to receive input from a user, perform arithmetic calculations and produce output on a screen. Topics included in this unit are:

Structure and Compilation of C/C++ programs

C/C++ identifiers, Variables, Declaration of Data Types and Constants.

Assignment Operations.

Arithmetic Operations

Data Type Conversions.

Basic Input/Output.

Comments and Indentation

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

ULO#1: Read, understand, and write simple C/C++ functions that declare variables, includes input and output from the user and perform arithmetic computations to solve problems, without syntax or logical errors. (CLO 1).
