# Unit 8: Overview

Introduction

In this unit we will review a few miscellaneous secure programming rules, outside of the topics previously discussed in class. Also, a summary of C/C++ constructs and a list of all secure programming rules studied in this course will be both presented in preparation for the Final Exam.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

ULO#1: Identify miscellaneous vulnerabilities in C/C++ programs and develop strategies to prevent those (CLO3 & CLO4).
