# Unit 5: Overview

Introduction

In this unit, C/C++ programs are presented as sets of functions collaborating with each other to perform a task. Having multiple functions adds complexity and it is the source of additional considerations to be considered when avoiding program vulnerabilities. Topics included in this unit are:

Function definition

Parameter Passing between Functions

Compilation of programs with Multiple Functions

Scope and Lifetime of Variables

Overloaded Functions

Random functions in C library stdlib.h

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

ULO#1: Develop C/C++ programs that use multiple functions to solve problems (CLO2).

ULO#2: Identify vulnerabilities in the use of multiple functions in a C/C++ program, and develop strategies to  prevent those (CLO3 & CLO4).
