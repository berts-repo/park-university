# Unit 6: Overview

Introduction

 In this unit we will review one the most powerful features in C/C++: pointers and their use. With pointers we will have direct control of a computer’s memory and with that we can do things that were not possible in previous units, like creating arrays that may change in size. With so much power come possible pitfalls. Programs falling in those may incorporate vulnerabilities that are exploitable by potential system attackers. Correct use of pointers by programs will eliminate this possibility. This unit will also study structures, a C/C++ construct to tie together various data types. Topics included in this unit are:

Pointer definition and Use

Pointer Arithmetic

Functions for Dynamic Allocation

Structures

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

ULO#1: Incorporate the use of Pointers, Dynamic Allocation, and Structures in C/C++ programs to solve problems (CLO2).

ULO#2: Identify vulnerabilities in the use of Pointers, Dynamic Allocation, and Structures in C/C++ programs and develop strategies to prevent those (CLO3 & CLO4).
