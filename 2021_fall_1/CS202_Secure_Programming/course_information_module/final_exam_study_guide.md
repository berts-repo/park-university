# Final Exam Study Guide

CS202 - Secure Programming

Exam Overview

The Final Exam for this course comprises most of the topics covered in the past weeks. In particular the students should be prepared for the following tasks:

Analyze a C/C++ program to find syntax and logic errors

Develop a C/C++ program from a given a problem description and construct with items such as strings, pointers, dynamic memory management, and file I/O to solve the problem.

Identify vulnerabilities in C/C++ programs that use constructs such as strings, pointers, dynamic memory management, and file I/O.

Develop practices and strategies for preventing vulnerabilities in C/C++ programs that use constructs such as strings, pointers, dynamic memory management, and file I/O.

Exam Format

The Final Exam will include approximately:

15 multiple-choice questions (worth 1 point each)

12 short-answer questions (worth 2 or 3 points each)

Total of 42 points

Most of the questions are similar to the ones given in quizzes throughout the course, so it is recommended that students carefully review these quizzes as preparation for the Final Exam.

Students are not required to memorize secure programming rules (SPRs) by heart, but be able to understand them if presented with them, and more importantly, be able to use them when appropriate.

While most of the topics covered in the course may be tested in the Final Exam in one way or another, students should possess core knowledge of C programing and secure programming techniques.

 

Exam Topics

The following is a (non-comprehensive) sample list of some of the knowledge that may be tested:

Unit 1

Basic structure of a C program, compilation and execution.

Correct structure of identifiers in C.

Variable definition: declaration and initialization.

Data types.

Unit 2

Correct use of control structures: conditional expressions, conditional statements and repetition statements.

Unit 3

Storage of integer and floating point values into variables:

Two complement’s format.

IEEE-754 format.

Considerations for data type conversions between unsigned and signed integers.

Unit 4

Character declaration, initialization and usage.

Array declaration and definition. Array navigation.

Finding the length of an array using sizeof.

String declaration and definition. String navigation.

Unit 5

Function declaration, function definition, function parameters.

Local, global, external, and static variables.

Unit 6

Pointer declaration, definition, de-referencing and usage.

Pointer arithmetic.

Dynamic allocation: proper use of malloc, calloc, realloc and free.

Declaration and handling of structures. Use of dot and arrow notations.

Unit 7

File opening, reading, writing, updating and closing.
