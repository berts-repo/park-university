# Unit 7: Overview

Introduction

In this unit we will review the use of files for input and output in C/C++ programs. Basic file operations are described together with some practical secure programming rules. Topics included in this unit are:

File Types

File Opening and Closing

Text Files Reading and Writing

Binary Files Reading and Writing

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

ULO#1: Incorporate the use of Files for Input and Output in C/C++ programs to solve problems (CLO2).

ULO#2: Identify vulnerabilities in the use of For Input and Output in C/C++ programs and develop strategies to prevent those (CLO3 & CLO4).
