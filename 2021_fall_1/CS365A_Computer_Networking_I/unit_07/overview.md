# Unit 7: Overview

Cisco CCNA Introduction to Networks v7 (ITN)

Module 11.1-11.5

Unit Learning Outcomes

In this unit, we will learn about the networking today, and the basic switch and end-device configuration.

ULO#1: Learn about the IPv4 address structure (CLO 3 & 4)

ULO#2: Learn about IPv4 subnetting scheme to efficiently segment the network  (CLO 3 & 4)
