# Unit 3: Overview

Cisco CCNA Introduction to Networks v7 (ITN)

Module 4 and Module 5

Unit Learning Outcomes

In this unit, we will learn about the physical layer, and the number systems.

ULO#1: Explain how physical layer protocols, services, and network media support communications across data networks (CLO 1 & 2)

ULO#2: Calculate numbers between decimal, binary, and hexadecimal systems (CLO 3)
