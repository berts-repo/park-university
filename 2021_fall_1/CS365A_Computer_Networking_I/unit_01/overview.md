# Unit 1: Overview

Cisco CCNA Introduction to Networks v7 (ITN)

Module 1 and Module 2.1-2.5

Unit Learning Outcomes

In this unit, we will learn about the networking today, and the basic switch and end-device configuration.

ULO#1: Explain the advanced technologies in modern networking (CLO 1 & 2)

ULO#2: Learn about the basic IOS command structure and implement initial settings such as device name and password (CLO 4)
