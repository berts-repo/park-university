# Unit 2: Overview

Cisco CCNA Introduction to Networks v7 (ITN)

Module 2.6-2.9, and Module 3

Unit Learning Outcomes

In this unit, we will learn the basic switch and end-device configuration, and protocols and models.

ULO#1: Implement IP addressing and default gateway parameters on a switch and end devices. (CLO 4)

ULO#2: Explain how network protocols enable devices to access local and remote network resources (CLO 1)
