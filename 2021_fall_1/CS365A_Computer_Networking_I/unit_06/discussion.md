Hello class,

This week I wanted to look into what was going on when we initially configure a cisco router/switch, and use the command line vty 0 4. The term vty stands for Virtual Teletype and is a configured virtual port, that gives simultaneous access to the router/switch with Telnet and SSH. 

The 0 4 represents the range of virtual ports to the router or switch. The max port range can be set to 16, which is represented by the command 0 15. This allows you to subdivide the network, to allow different areas to virtually access the router/switch simultaneously. 

The line can be configured to have a single password for all-users, or configured to have individual passwords, that are user-specific. This can be configured locally or setup through an authentication server.
