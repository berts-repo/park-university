# Unit 5: Overview

Cisco CCNA Introduction to Networks v7 (ITN)

Module 8 and Module 9

Unit Learning Outcomes

In this unit, we will learn about the network layer and address resolution.

ULO#1: Explore how routers use network layer protocols and services to enable end-to-end connectivity (CLO 1 & 2)

ULO#2: Examine how ARP and ND enable communication on a network (CLO 1 & 2)
