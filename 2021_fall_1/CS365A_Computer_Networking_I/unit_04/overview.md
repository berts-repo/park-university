# Unit 4: Overview

Cisco CCNA Introduction to Networks v7 (ITN)

Unit Learning Outcomes

In this unit, we will learn about the data link layer, and Ethernet switching.

ULO#1: Explain how media access control in the data link layer supports communication across networks (CLO 1 & 2)

ULO#2: Examine how Ethernet operates in a switched network (CLO 1 & 2)
