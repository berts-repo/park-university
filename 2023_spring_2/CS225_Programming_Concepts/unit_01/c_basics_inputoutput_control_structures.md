# Unit 1: C++ Basics, Input/Output, Control Structures

Introduction

This unit's main goal is to get familiar with the fundamentals of C++. Students should get their compilers up and running as quickly as possible. Particular attention in this unit is paid to the fundamental properties and syntax of C++ and input/output.  Control structures in C++ are almost identical as in Java, except for how Boolean values are handled.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

ULO 1: Create programs with stream input/output for both console and text files. (CLO 1)

ULO 2: Create programs using control structures (if-else, switch, loops) (CLO 1)

ULO 15: Analyze the syntax and execution of C++ programs. (CLO 5)
