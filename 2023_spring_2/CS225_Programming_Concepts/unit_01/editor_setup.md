# Editor Setup

Set-up your editor not to use tabs

Requirements

Set up your editor to emit spaces (also known as soft tab) when you hit the tab key.

Set up your C++ editor/IDE to use spaces instead of tab for indentation, and indent 2 spaces at a time.

 

Microsoft Visual Studio

Launch Microsoft Visual Studio

Go to Tools Menu and select Options

Select Text Editor 

Select C/C++

Select Tabs and then adjust the settings

Tab size: 2

Indent size: 4

Check the radio button for Insert spaces

 

Visual Studio Code

VS Code’s  [link: https://code.visualstudio.com/docs/getstarted/settings#_default-settings] default setup may have done this for you.
