Q0. If you declare an array double list[] = {3.4, 2.0, 3.5, 5.5}, what's the highest index in this array?

Q0. The answer is 3. Here the length of the array is determined by the number of initial values in the braces.  There are 4 elements in the declaration thus the size of this array is 4. Since the index starts at 0, the highest index is 3 (4 minus 1).
