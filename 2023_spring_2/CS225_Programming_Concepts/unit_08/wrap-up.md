# Unit 8: Wrap-Up

Introduction

You will review what we have covered in this course and prepare for the final exam.  The final exam is comprehensive and covers material from Units 1-7. 

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Create programs with stream input/output for both console and text files. (CLO 1)

Create programs using control structures (if-else, switch, loops) (CLO 1)

Create programs that require the use of programmer-defined functions. (CLO 2)

Select pass-by-value, pass-by-reference, or pass-by pointer for a function parameter. (CLO 2 and 3)

Construct generic code with C++ templates. (CLO 2)

Create programs that require the processing of elements of one or multiple arrays or container objects. (CLO 3)

Contrast C-string and C++ string class. (CLO 4)

Construct programs that use C-string or C++ string class. (CLO 4)

Construct programs using pointers and dynamic memory allocation. (CLO 3)

Compare C language structs with C++ classes. (CLO 1)

Create programs that use C language structs. (CLO 1)

Create classes including the use of overloaded functions/operators, and constructors (CLO 1)

Compare the three types of inheritance: public, protected, private. (CLO 1)

Create a hierarchy of classes based on inheritance and polymorphism (CLO 1)

Analyze the syntax and execution of C++ programs. (CLO 5)
