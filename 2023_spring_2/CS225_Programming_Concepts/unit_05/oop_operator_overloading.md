# Unit 5: OOP; Operator Overloading

Introduction

This unit covers Object-Oriented Programming (OOP) in C++. The concept and principles are the same as in Java - a class encapsulates both data and operations (the #1 principle of OOP – encapsulation). We will also look into operator overloading, which extends the definition of an operator to work with a user-defined data type.  Operator overloading is one of C++’s most appealing attributes, which contributes to C++’s extensibility.  

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

ULO 12: Create classes including the use of overloaded functions/operators, and constructors (CLO 1)

ULO 15: Analyze the syntax and execution of C++ programs. (CLO 5)
