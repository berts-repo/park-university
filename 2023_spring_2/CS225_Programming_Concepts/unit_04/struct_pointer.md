# Unit 4: Struct; Pointer

Introduction

This unit introduces another structured data type, called a struct (some languages use the term “record”).  A struct groups related items of different types while an array manipulates a group of of related items of a same data type. A struct is like a Java class, except that all members are public and generally doesn't contain member functions (member methods in Java).

We will also cover the pointer data type in this unit. A pointer type describes an object whose value provides a reference (internal address) to an entity of the referenced type. You will first learn how to declare pointer variables (or pointers, for short) and manipulate the data to which they point. Later, you will use these concepts when you study dynamic arrays and dynamic data members of a class (chapter 12. Will be covered in Unit 6).
