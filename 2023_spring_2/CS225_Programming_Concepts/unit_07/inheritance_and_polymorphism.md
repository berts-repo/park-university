# Unit 7: Inheritance and Polymorphism

Introduction

This unit discusses two of the three main principles of object-oriented design: inheritance and polymorphism (we already covered encapsulation in Unit 5).  Inheritance allows us to create new classes from existing classes. A new class built via inheritance (often referred to as subclass, child class, or derived class) is a more specific version of the existing class (known as super class, parent class, or base class). Polymorphism refers to the ability for the same code to be used with different types of objects and behave differently. The concepts and principles of inheritance and polymorphism are the same as in Java, only that the syntaxes are different.
