Software

Title: A suitable C++ compiler/IDE for your computer

Adoption: Required

Please check with your course instructor which C++ compiler will be used for grading programming assignments.  Make sure the one you use is compatible. Instructions for acquiring the software

 [link: https://www.visualstudio.com/vs/] Microsoft Visual C++ is recommended.

The free Community version is fine. See the  [link: https://docs.microsoft.com/en-us/visualstudio/install/install-visual-studio?view=vs-2019] Install Visual Studio (installation guide) from Visual Studio Doc.  Choose the "Desktop development with C++" workload at step 4 - choose workloads.  Previous versions of Visual Studio such as 2017, 2015, 2013 works too.

Additional Visual Studio Doc:  [link: https://docs.microsoft.com/en-us/cpp/build/vscpp-step-1-create?view=vs-2019] Create a C++ console app project. Continue into the "Build and run a c++ project" link under "Next Steps" or from the navigation menu on the left-hand side of the page.

You may access Visual Studio via Park’s virtual lab without any installation. See  [link: https://parkuniversity.freshdesk.com/support/solutions/articles/6000177797-virtual-lab-applications] Park Virtual Lab Applications for details. It may be slower than using a C++ compiler installed on your computer.     

Any other C++ compiler/IDE, such as   [link: https://code.visualstudio.com/] Visual Studio Code (Help:  [link: https://code.visualstudio.com/docs/languages/cpp] C/C++ for Visual Studio Code),  [link: http://www.codeblocks.org/downloads/binaries] Code::Blocks,  [link: http://www.eclipse.org/downloads/packages/] Eclipse for C/C++, XCode (Mac only) are fine. Mac/UNIX/Linux users can also directly use the GNU C++ compiler g++ from the command line.

Online compilers such as  [link: http://cpp.sh/] C++ Shell may be handy when testing snippets of code.
