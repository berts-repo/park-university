Course Details

Title: Programming Concepts

Course Description: This course continues the development of the programming and problem solving skills introduced in CS219. Programming concepts will be put into practice by using C for programming projects. Since C is so similar to Java and since students should already know Java from their prerequisite courses this course will cover C basics (control constructs operators data types functions) very quickly. More time will be spent on those features of C that differ from Java. For example more time will be spent on pointers object-oriented programming techniques and operator overloading. 

Credit Hours: 3:0:3

Prerequisite Courses: CS219 (Grade of C or better)

Core Learning Outcomes

At Park University, course objectives are called course learning outcomes (CLO). The specific outcomes for your course will always appear in your syllabus. (Note: You can view the approved Core Learning Outcomes for any course offered at Park University at  [link: https://app.park.edu/syllabus/clo.aspx] https://app.park.edu/syllabus/clo.aspx.) Undergraduate Core Learning Outcomes are approved by the approved by the academic department and the University Assessment Committee.

At the conclusion of the course, learners will be able to:

Given a problem description, implement a C++ program that uses inheritance and operator overloading.

Implement functions with call-by-value, call-by-reference, and default parameters.

Use pointers, references, and both 1- and 2-dimensional arrays.

Compare C-style arrays of characters with C++ strings.

Analyze a C++ program to find syntax and logic errors.

The map below shows the conceptual dependencies of the 15 unit learning outcomes. Input/Output concepts (ULO 1) are used in control structures (ULO 2), which then branch out into functions (ULOs 3, 4), arrays (ULO 6), and structs (ULO 10, 11). This information will be applied to classes (ULO 12). Functions require knowledge of templates (ULO 5), classes, and inheritance (ULOs 13, 14). Arrays require knowledge of pointers (ULO 9), C-string, and C++ string class (ULOs 7, 8). 

 

Course Structure

This section shows you how the learning process of this course is structured.

Each Unit contains a Unit Overview which contains the learning outcomes, activities, and assessments for that unit. You should always read and print out each Unit Overview.

Unit structure of the course:

This course is split into eight units.

The unit starts on Monday at 12 am CST and ends on Sunday at 11:59pm CST (CST is the Central Standard Time zone, the time zone of the Parkville Campus.)

Course Attendance requirements will require all students to log into the course at least once during each unit.

Course Participation requirements will require you to log in more frequently and are outlined in detail in the course syllabus. (see below for additional information on participation requirements and grading criteria.)

Refer to the Course Schedule for more information.

Overview of the Course Learning Activities

Each unit, you will have regular learning activities:

Reading - required readings from the text. These are listed on the unit overview page. 

Refer to Grading and Assignments for more detailed information on the learning activities and assessments.

Overview of the Course Assessments

Each unit, you will have regular assessments:

Discussions (115 points): Participate in the unit discussions according to the Discussion Rubric at the bottom of the page. There are unit graded discussions from units 1-8.

Programming Homework (420 points): Please print and read the assignment directions in each unit. There are one or two programming assignments in each unit. These assignments will always be due on Sunday. A detailed grading criterion is provided with each assignment.

Software Requirements

Use an integrated development environment (IDE) to build and test C++ programs.

Submit assignments via LMS course tools.

Quiz (105 points):  Complete a quiz during units 1- 7.
The quizzes will contain T/F and multiple choice questions.

There will be around 15 questions on the quizzes which will all be taken online in Canvas. You will have up to 5 attempts and 30 minutes for each attempt for each quiz.

Final Exam (360 points): Complete a final exam during unit 8. The exam will contain short answer, multiple choices, and coding questions. The exact number of questions could change each semester. You will be required to read and debug code, and create a program.
