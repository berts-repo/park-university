# Unit 3: Arrays and Strings

Introduction

C++ data types fall into three categories: simple, structured, and pointers. A data type is called simple if variables of that type can store only one value at a time. In contrast, in a structured data type, each data item is a collection of other data items. The first structured data type that we will discuss is an array. The concept of C++ array is the same as in Java.  We will also examine c-strings, a '\0' terminated character array used in C/C++ to handle string literals. We will see how it differs from the string class.
