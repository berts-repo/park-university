# Unit 6: Classes with Dynamic Data; Class Template

Introduction

In this unit we will build classes with dynamic data members.  Dynamic data is implemented as pointer variables referencing data dynamically allocated during execution.  A class with dynamic data should always include a copy constructor, overload the assignment operator, and a destructor.  We will also discuss how to define a generalized class using class template.  Finally, we will take a look at the vector class in the Standard Template Library, which is both a class template and a class with dynamic data.  The vector class in C++ is very similar to the ArrayList class in Java.
