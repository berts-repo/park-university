# Unit 7: Overview

Writing Web Programs with JavaScript

Introduction

Students will learn how to create a form and dynamically update a table. In this unit students will take the next step and work with a class, that will allow the data to be sorted each time the table is updated. 

Topics include: 

Working with classes, properties and methods.

Create forms with HTML and process the form with JavaScript.

Style forms and output with classes and CSS style rules 

Use of external JavaScript files.  

Implement classes with constructors, properties, and methods.

Implement event handlers where the event object helps to process the event.

Learn how to add event handlers dynamically using the addEventListener method.

Use an array to store multiple data items using a common name for all of them.

Learn the syntax for instantiating an array and looping through an array’s elements.

Use arrays to dynamically sort and update content.

Unit Learning Outcomes

At the conclusion of the unit, learners will be able to:

Demonstrate creativity and problem-solving skills.

Analyze web programs in order to test, debug, and improve them.

Evaluate web pages and web programs to ensure that they use proper coding conventions and documentation.

Create web pages and web programs that use: HTML5, cascading style sheets, dynamic HTML, JavaScript, forms with controls, and Canvas.
