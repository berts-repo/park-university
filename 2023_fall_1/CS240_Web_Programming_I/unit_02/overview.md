# Unit 2: Overview

Working with HTML and CSS

Introduction

This unit you will work with HTML and CSS to create a web page. They will create a page that uses ordered, unordered and nested lists, heading, figures and figure captions and CSS style rules. 

Topics include: 

CSS style rules using an external style sheet

Inline style rules to format the contents and layout of your page

Selectors such as child and descendant selectors

Classes and how they can be used to format web page content

Overrule external style rules with inline style rules

Style properties for color, font, text, border, padding and margin

Comments, indentation and proper naming conventions

Create a navigation menu using an unordered list

Ordered and nested lists

Figures which display images and a caption

Organizational elements like section, article, and aside, headers and footers

Navigation bar using nav and a elements

Unit Learning Outcomes

At the conclusion of the unit, learners will be able to:

Demonstrate creativity and problem-solving skills.

Analyze web programs in order to test, debug, and improve them.

Evaluate web pages and web programs to ensure that they use proper coding conventions and documentation.

Create web pages and web programs that use: HTML5, cascading style sheets, dynamic HTML, JavaScript, forms with controls, and Canvas.
