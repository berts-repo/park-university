Course Details

Title:  CS 240 - Web Programming I

Course Description: 
This course provides an introduction to the various languages, tools and programming techniques used for client-side programming on the World Wide Web. Concepts will be put into practice by using HTML, cascading style sheets, and JavaScript. Due to the particularly dynamic nature of the web environment, course content will change as appropriate.

Credit Hours: Lecture Hours: 3 Lab Hours: 0 Credits: 3

Prerequisite(s): CS 151 or CS 152.

Note: In this class we will spend a significant amount of time on HTML, CSS and JavaScript. 
Java and JavaScript are different technologies. 
In this class we are only working with JavaScript, not Java!

Core Learning Outcomes

At the conclusion of the course, learners will be able to:

Demonstrate creativity and problem-solving skills.

Analyze web programs in order to test, debug, and improve them.

Evaluate web pages and web programs to ensure that they use proper coding conventions and documentation.

Create web pages and web programs that use: HTML5, cascading style sheets, dynamic HTML, JavaScript, forms with controls, and Canvas.

How much time will this course take me, really?

This is a challenging course. I will help you as best I can. However, you will have to put your effort and time into the course to complete all of the required readings and learning activities in order to be successful in this course. You cannot possibly learn web programming unless you actually program for the web. 

Therefore it is imperative that you keystroke your own work and solve your own problems. While web sites can use useful as a reference you will not be allowed to copy code from the web or other sources. I have tried to include lessons, notes that will help guide you through the readings in the book.

The amount of new material and accelerated 8-unit format requires you to be prepared, read all the materials, and practice, practice, practice! We expect you to be actively participating in discussions and learning activities.

Course Assignments.

Discussion Questions (17% - Unit 1 - 8 is worth about 20 points. ) - There will be discussion questions in Units 1- 8.  Initial posts are due by 11:59 p.m. Thursday, CT.  After posting your answers, begin making 2 or more replies to other students' posts due by 11:59 p.m. Sunday, CT.

Homework (48% - 60 points each unit) - In Units 1-8, you will have a homework assignment that involves creating a web page or program that is due by 11:59 p.m. Sunday, CT. You may not use third party code in your homework except what is provided to you in the assignment. 

Proctored Final Exam (35% - 350 points) - In Unit 8, you will complete a final exam which is cumulative. The final exam consists of two parts. The first part you will answer multiple choice and short answer questions about the concepts and programming you learned during the course.
