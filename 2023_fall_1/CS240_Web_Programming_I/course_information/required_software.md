# Required Software

Which Browser Do I Use?

You will need to have a web browser to preview your web pages. There may be slight differences in your web pages depending on which browser you use. It is best to preview your web pages in at least two of the major browsers. 

Major browsers include Google Chrome ( [link: https://www.google.com/chrome/] Download Google Chrome) and Firefox ( [link: https://www.mozilla.org/en-US/firefox/new/] Download Firefox), Microsoft Edge and Apple's Safari browser. Microsoft Edge is installed by default on Windows 10 systems. Safari is the web browser installed by default on Mac computers. 

Note that this course primarily uses Microsoft Edge in the demonstrations the default browser.  

Which Web Page Editor Do I Use?

Creating web pages in this course, can be accomplished using a text editor like Notepad or Notepad++, or a web page editor such as Brackets, Visual Studio, Visual Studio Code, Visual Studio for the Mac, and many other editors. 

Students will be responsible for conforming to the assignment procedures. Your instructor will help you with how to create your programs using HTML, CSS and JavaScript. It is usually recommended to update your operating system and back up your system and files before installing new software. Instructors cannot provide technical support for your individual computer systems. 

To develop web pages created only with HTML you could just type in the code in a text editor. But having an editor will help you identify your errors. Some editors provide color coding of tags and attributes, separate from the text. This feature can help you to identify errors like missing quotation marks or incorrect closing tags.

Some editors have a full development environment which can help you manage your folders and files. Some editors provide tools to help with verifying the syntax of your code or debugging environments. These features can be helpful to you as write the code and debug your program.  

However, some editors may include other features such as templates or sample web pages. Similarly, some editors will provide a graphical user interface (GUI) where you can drag and drop controls on the page. For example, in Dreamweaver the editor can be used to create the mouseover effect without writing any code. Please do not use these features unless given instructions from your instructor. Using these GUI features are not permitted in your assignments. 

Some editors may insert obsolete or deprecated code, depending on the editor you choose. Please make sure to use editors that support HTML 5. 

What is required is that when you create your code in your web programs, you enter the code manually. The purpose of the course is to learn how to write your own code. Do not use any of the graphical editing programs or built-in tools that insert the code for you. You will not receive credit for code that you did not write. You are responsible for coding all of your work manually.

Virtual Machines for Mac Users   

Students using Mac computers who want to run the full Windows version of Visual Studio locally on their own computer, technically have one other choice.

If you are a Mac user, and really want so use the Windows version of Visual Studio on your own computer, there is a way to do this locally. However, this is not for beginners and requires additional software. So why mention it? Once in a while a student who is working in industry wants to run Visual Studio. Visual Studio is the tool most often to create web server side programs with ASP. We don't cover anything in that 

Mac users 'could' run any Windows versions of Visual Studio if they run them within a virtual machine using virtualization software such as  [link: https://www.vmware.com/products/workstation-pro.html] VMWare Workstation,  [link: https://www.parallels.com/products/desktop/] Parallels or Virtual Box.  [link: https://www.virtualbox.org/] Virtual Box is free. You would also need to download a copy of Windows from Imagine and install that into the virtual machine before you would install Visual Studio. 

While technically possible, if you do not have experience using a virtualized machine, do not choose this option. However, this option is unnecessary for this course. Given that the online courses are only 8 weeks long, there are much easier solutions for students who use Mac computers.
