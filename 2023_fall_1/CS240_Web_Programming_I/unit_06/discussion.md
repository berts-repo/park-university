CH 10 Review Question #23 (sec. 10.16)

What does it mean if a list box object’s selectedIndex property is -1?

The selectedIndex method is used to reset the list box, so there is no longer an option selected. We can reset the list box by calling the statement on some variable:

variable.selectedIndex = -1;

CH 10 Exercises  #6 (sec. 10.9) 

The web page code in the prior exercise generates the factorial for a user-entered number. Why would it be inappropriate to replace the while loop with a for…of loop.

It would be inappropriate to use the for...of loop because ‘num’ is just a single user-entered number. A for…of loop is implemented when working with a collection. The for…of loop allows us to iterate over data structures like arrays and strings. (JavaScript For Of, 2023) The statement will look like this:

for (let variable of collection)
for (let letter of alphabet){
      // Do something
}
