# Unit 6: Overview

Writing Web Programs with JavaScript

Introduction

This unit you will work with HTML and CSS to create a web page. You will create an invitation that will allow the user the ability to create a dynamic invitation on the fly. 

Topics include: 

Create an interactive program with HTML, CSS, JavaScript and DOM

Document your program using comments, indentation and proper naming conventions

Forms, form controls, HTML validation and processing forms

Programming logic such as for loops, while loops

External JavaScript files 

Working with functions and event handlers

Dynamically building HTML and form controls using JavaScript. 

Converting numbers to integers and formatting dates and numbers

Populating form controls such as dropdown lists with data

Unit Learning Outcomes

At the conclusion of the unit, learners will be able to:

Demonstrate creativity and problem-solving skills.

Analyze web programs in order to test, debug, and improve them.

Evaluate web pages and web programs to ensure that they use proper coding conventions and documentation.

Create web pages and web programs that use: HTML5, cascading style sheets, dynamic HTML, JavaScript, forms with controls, and Canvas.
