an element

a text item between an element’s start and end tags

an attribute within an element

Chapter 8: Exercise 6:

Why is it good to use var for same-named variables that are in different functions?

The use of var is great for readability, by declaring var before a variable it makes it easier to determine what a variable is. Declaring the variable is also common syntax for different programming languages. Besides the readability aspect, declaring var also makes the scope of the variable more local, this means that if you have two functions that use the same variable name, it will only use the variable in that particular function.
