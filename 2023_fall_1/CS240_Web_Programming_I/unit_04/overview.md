# Unit 4: Overview

Working with HTML and CSS

Introduction

After successfully completing this unit, from your readings, lectures and assignments, you will have learned how to:

Topics include: 

How to use CSS 3 to enhance the web page content.

How to create a form with HTML.

How to use JavaScript to modify the styles of the form dynamically. 

How to use JavaScript to modify the style rules dynamically.

How to document your program using comments, indentation and proper naming conventions.

How to validate the form with HTML validation and process a form using JavaScript

Unit Learning Outcomes

At the conclusion of the unit, learners will be able to:

Demonstrate creativity and problem-solving skills.

Analyze web programs in order to test, debug, and improve them.

Evaluate web pages and web programs to ensure that they use proper coding conventions and documentation.

Create web pages and web programs that use: HTML5, cascading style sheets, dynamic HTML, JavaScript, forms with controls, and Canvas.
