# Unit 3: Overview

Working with HTML and CSS

Introduction

This unit will go further into how to use CSS and HTML to manually layout a web site. Students will use their page to insert and format a table. 

Topics include:

Create and format tables using HTML and style rules.

Formatting a page using HTML and CSS. 

Creating a menu with hyperlinks in an aside. 

Inserting images into a header with CSS. 

Creating hyperlinked thumbnail images. 

Use JavaScript to create an interactive program 

Image map, iframes, forms, proper documentation and coding conventions. 

Unit Learning Outcomes

At the conclusion of the unit, learners will be able to:

Demonstrate creativity and problem-solving skills.

Analyze web programs in order to test, debug, and improve them.

Evaluate web pages and web programs to ensure that they use proper coding conventions and documentation.

Create web pages and web programs that use: HTML5, cascading style sheets, dynamic HTML, JavaScript, forms with controls, and Canvas.
