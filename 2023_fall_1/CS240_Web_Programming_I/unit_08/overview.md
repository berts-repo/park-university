# Unit 8: Overview

Writing Web Programs with JavaScript

Introduction

Students will learn how to create a use the Canvas element and JavaScript to build an interactive application. 

Topics include: 

Create a graphical scene using the canvas object. drawing tools and JavaScript.

Creating an interactive canvas program with JavaScript

Document your program using comments, indentation and proper naming conventions

Unit Learning Outcomes

At the conclusion of the unit, learners will be able to:

Demonstrate creativity and problem-solving skills.

Analyze web programs in order to test, debug, and improve them.

Evaluate web pages and web programs to ensure that they use proper coding conventions and documentation.

Create web pages and web programs that use: HTML5, cascading style sheets, dynamic HTML, JavaScript, forms with controls, and Canvas.

Assignments

Discussion  

In the discussion you will work with your classmates to review the end of the chapter review questions and programming assignments. You will need to post a reply to 2 students.

Homework  

Students will create a web page using Visual Studio and HTML that includes a JavaScript that will provide interactivity within the web page. The students will use the drawing tools to draw on the canvas and animate an image. 

Final Exam

The Final Exam is in this unit! Please review the Review Sheet provided in Unit 7. Please note that you will also be tested on all of the material from Units 1 - 7 on the final exam. Because discussion and homework is not due until the last night of the class, final grades will be posted two days after Unit 8 has closed. Access final exam through Honorlock on the Canvas menu.
