Review Question: #12

Who designed the HTTP protocol?

Tim Berners-Lee designed the HyperText Transfer Protocol (HTTP), between 1989-1991.Berners-Lee then implemented this protocol and created a browser that could click around the internet called TheWorldWideWeb with the use of HTML as the markup language, which he also designed at the same period. (MozDevNet)

 

Exercise Question: #3

Explain all the parts of a URL. For example, in https://mars.jpl.jasa.gov/msl/images/PIA16082_Mitrofanov1f-thm.jpg, what is each of the fallowing?

https - HyperText Transfer Protocol Secure, which is the protocol that communicates between the webserver and browser. 

mars - The subdomain which is the name of the system that the web server is hosted on.

jpl.jasa.gov - The domain which is the address to find a webpage on the internet.

msl/images - The path which is the file hierarchy of where things are stored on the web server.

PIA16082_Mitrofanov1f-thm.jpg - The image file on the web server that is being accessed.
