# Unit 1: Overview

Introduction to Web Programming

This unit you will learn how to create basic web pages with HTML using HTML elements and attributes.  

Topics include: 

How to install and configure the required software. 

How to create a basic web page using HTML using a web page editor.  

How to format content with HTML elements and attributes. 

How to debug HTML code and use validation and syntax checking tools. 

How to document your HTML code using comments, indentation and proper naming conventions.  

Unit Learning Outcomes

At the conclusion of the unit, learners will be able to:

Demonstrate creativity and problem-solving skills.

Analyze web programs in order to test, debug, and improve them.

Evaluate web pages and web programs to ensure that they use proper coding conventions and documentation.

Create web pages and web programs that use: HTML5, cascading style sheets, dynamic HTML, JavaScript, forms with controls, and Canvas.
