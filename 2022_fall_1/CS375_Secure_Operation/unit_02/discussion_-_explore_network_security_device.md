Network Access Control (NAC) allows visibility, and information about any device that connects to the network. It allows the filtering out of unwanted devices and connections, that don't meet the current policies or specified parameters. It can deny someone access to secure documents when accessing them remotely or from a unsecure location. It can also verify software and OS versions, and quarantine the end device until the appropriate updates are made. Not only does it quarantine outdated software, it also can also isolate a nodes that are susceptible to 0-day threats, which this information is often shared between vendors. (Cisco 2021) It provides different security device tools like a VPN.

NAC doesn't always work with other security measures. When working with virtualization it becomes very hard to use NAC software, virtual servers can move around a data center and dynamically allocated addressing information. NACS also struggle with virtualization because the VMs can be in a rest state and the NAC wont necessarily recognize critical updates from this state. (What is network access control?)'

There are many different vendors for NAC software but CISCO is one of the most reputable, with its ISE solution. Some other popular vendors include The Forescout Platform and Aruba ClearPass Policy Manager. (Gartner 2022)
