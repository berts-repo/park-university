# Unit 3: Overview

Module 15, 16, and 17

In this unit, we will learn the importance of traffic monitoring and how it is conducted. This is followed by an in-depth discussion of attacks on various network protocols, including IP, TCP, ARP, DNS, DHCP, etc.

 [link: #ULOs] Click for Unit Learning Outcomes

 

ULO #1: Apply and describe various network monitoring techniques and tools. (CLO 4)

ULO #2: Explain how TCP/IP vulnerabilities enable network attacks. (CLO 3)

ULO #3: Explain how common network applications and services are vulnerable to attack.(CLO 3)
