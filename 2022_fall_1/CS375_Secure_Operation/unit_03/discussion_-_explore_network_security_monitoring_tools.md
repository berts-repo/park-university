snort is an open-source IPS who was bought by Cisco Talos in 2013, that can drop malicious traffic from the network. (Snort 101) Snort has a packet sniffer that detects, monitors, and alerts from different attack vectors, and then it can send a report through a different channel, e.g. a cell phone. (Berman 2021) Snort can be ran in three different modes: Sniffer Mode, Packet Logger, and Network Intrusion Detection System (NIDS). Snort has been integrated into many different SEIMS, IDS, and IPS, but the most common alternative to Snort is Suricata. (Berman 2021)

Sniffer mode - which simply reads the packets off of the network and displays them for you in a continuous stream on the console (screen).
Packet Logger mode - which logs the packets to disk.
Network Intrusion Detection System (NIDS) mode - which performs detection and analysis on network traffic. This is the most complex and configurable mode.
