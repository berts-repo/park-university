Module 1-4

In this unit, we will learn basic concepts about cyber-attacks, including who, what, and why of cyberattacks. In addition, we will learn basic function and techniques in Security Operations Centers (SOC), the department in each organization for defending cyberattacks. We will also learn administration basics for the Windows Operating System, including how the operating system works and the tools used to secure Windows endpoints

 [link: #ULOs1] Click for Unit Learning Outcomes

ULO #1: Identify various types of attacks, threat actors, and threat impacts (CLO 3)

ULO #2: Explain the function, composition, process, and technologies of the Security Operations Center (SOC). (CLO 1)

ULO #3: List Windows OS security recommendations. (CLO 2)

ULO #4: Paraphrase Windows architecture and operations. (CLO 2)

ULO #5: Operate on Windows OS to configure and monitor user accounts, network, and the security. (CLO 4)

Module 11-14

In this unit, we will cover the basic operation of network infrastructures, including wired and wireless networks, network security, and network designs. Specifically, we will learn a set of network devices, such as routers, switches, and network security infrastructure, such as firewalls, routers, switches, etc. Additionally, we will see a variety of tools and methods threat actors use to launch attacks.

 [link: #ULOs] Click for Unit Learning Outcomes

 

ULO #1: Describe various network communication devices. (CLO 2)

ULO #2: Analyze the setup of the network security infrastructure. (CLO 2)

ULO #3: Analyze the network topology setup. (CLO 2)

ULO #4: Explain, demonstrate and analyze various network attacks. (CLO 3)

Module 26, 27, & 28

 

In this unit, we will learn how network security alerts are reported, evaluated, escalated, and preserved as evidence. We will also cover incident response and handling models and procedures. These include the Cyber Kill Chain, the Diamond Model, the VERIS schema, and NIST guidelines for the structure of Computer Security Incident Response Teams (CSIRTs) and processes for handling an incident.

 [link: #ULOs] Click for Unit Learning Outcomes

 

ULO #1: Explain the process of evaluating alerts(CLO 4)

ULO #2: Analyze various security alerts. (CLO 4)

ULO #3: Explain digital forensic techniques. (CLO 4)

ULO #4: Explain multiple incident response models and techniques. (CLO 4)

ULO #5: Explain multiple incident handling models and techniques. (CLO 4)
