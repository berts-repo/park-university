# Unit 4: Overview

Module 18, 19, 20, and 21

In this unit, we will learn different approaches to network security defense, access control methods, and the various sources cybersecurity analysts rely on for threat intelligence. Additionally, we will cover various cryptography technology and public key infrastructure (PKI) used for the network security.

 [link: #ULOs] Click for Unit Learning Outcomes

 

ULO #1: Describe and analyze network defense techniques and policies (CLO 4)

ULO #2: Explain access control as a method of protecting a network. (CLO 4)

ULO #3: List  various intelligence sources to locate current security threats. (CLO 4)

ULO #4: Explain and apply different cryptography algorithms. (CLO 4)

ULO #5: Explain and analyze Public Key Infrastructure (PKI) trust system. (CLO 2)
