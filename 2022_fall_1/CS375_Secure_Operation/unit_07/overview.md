# Unit 7: Overview

Module 26, 27, and 28

In this unit, we will learn how network security alerts are reported, evaluated, escalated, and preserved as evidence. We will also cover incident response and handling models and procedures. These include the Cyber Kill Chain, the Diamond Model, the VERIS schema, and NIST guidelines for the structure of Computer Security Incident Response Teams (CSIRTs) and processes for handling an incident.
