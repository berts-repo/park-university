# Unit 1: Overview

Module 1-4

In this unit, we will learn basic concepts about cyber-attacks, including who, what, and why of cyberattacks. In addition, we will learn basic function and techniques in Security Operations Centers (SOC), the department in each organization for defending cyberattacks. We will also learn administration basics for the Windows Operating System, including how the operating system works and the tools used to secure Windows endpoints, as well as Linux Operating System, including how to perform basic Linux operations and security-related tasks.

 [link: #ULOs] Click for Unit Learning Outcomes

ULO #1: Identify various types of attacks, threat actors, and threat impacts (CLO 3)

ULO #2: Explain the function, composition, process, and technologies of the Security Operations Center (SOC). (CLO 1)

ULO #3: List Windows OS security recommendations. (CLO 2)

ULO #4: Paraphrase Windows architecture and operations. (CLO 2)

ULO #5: Operate on Windows OS to configure and monitor user accounts, network, and the security. (CLO 4)

ULO #6: Perform common commands on Linux OS. (CLO 4)

ULO #7: Apply basic administration on Linux server (CLO 4)
