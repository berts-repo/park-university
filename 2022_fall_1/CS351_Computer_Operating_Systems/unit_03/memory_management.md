# Unit 3: Memory Management

Overview

Memory management is crucial in modern multitasking operating systems. This chapter introduces you to the basic elements in memory management and several memory management algorithms.

Learning Objectives

Upon completing this unit, students will:

Classify the requirements of memory management

Contrast the concepts of paging and segmentation

Differentiate the concepts memory partitioning

Define virtual memory and examine how it is used

Compare and contrast memory management in Linux and Windows
