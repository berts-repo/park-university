the amount of RAM your computer has a direct correlation to the performance. The more RAM means the less the processor has to fetch data from the hard-drive, which is drastically faster, as it takes a long time to read/write to a hard-drive. With less reading/writing to the hard-drive, means it also leaves the illusion that you can run more programs at once, as there is enough address space for all the different processes.

Windows 11 and Ubuntu both require 2 cores with a minimum of 4gb of RAM, and Windows 10 only requires 2gb. There is a reason they call these "minimum requirements," but it almost isn't enough: Google Chrome alone requires about 1gb ram, but if you run more then one instance or start opening a lot of tabs you can easily occupy 2gb of RAM. The Adobe Premier suite requires 8gb of RAM and 2gb of VRAM.  So, with modern multiprocessing computers the minimum requirements are not ideal for modern computing.
