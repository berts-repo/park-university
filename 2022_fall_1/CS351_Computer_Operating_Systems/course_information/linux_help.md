# Linux Help

This page is a Linux help repository for Linux hands-on projects in this course. Information in this page will be updated when a site is found to be helpful for students to understand and complete their projects.

When in doubt, use "man" command. For example, "man proc" for information about /proc, "man pthread_create" for information about pthread_create.
