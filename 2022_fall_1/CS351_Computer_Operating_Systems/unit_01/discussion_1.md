my favorite thing about Linux compared to Windows is that it is free and open-sourced, but that's not the technical differences between the two. Linux was historically produced for the developers, while Windows 10/11 was developed for commercial use and the user experience. Linux also has better access to the underlining hardware through system calls and it's shells, where Windows relies Win32 API to access the hardware. Linux has a monolithic kernel, which means that the entire OS operates inside the kernel, where Windows 10 and 11 uses a hybrid kernel, which uses a combination of a micro and monolithic kernels. Windows GUI operates inside of the kernels, and a Linux GUI is an application. Another primary application for Linux is a shell, and the most common is the bash shell. The BASH shell allows the users to run commands and calls, and even write these into scripts to process in sequential order, giving a lot more flexibility to developers and engineers.
