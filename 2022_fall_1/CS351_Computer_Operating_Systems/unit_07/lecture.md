# Unit 7: Lecture

Chapter 9: Security

As we have seen in the past units, an operating system controls the operations of a computer. It is imperative to have a strong and secure operating system. However, in most operating systems, security does not build into the operating system from the beginning. When the security is not considered in the beginning, attackers can find holes to break into an operating system, thereby gaining control of the computer system. In these systems, secure components are added to the operating system as holes are discovered.

Overall, in operating systems, there are four kinds of security services:

User Authentication: User authentication is used to determine the legitimacy of a user.

Data Confidentiality: With data confidentiality, an attacker can not understand the message even if he can obtain the message.

Data Integrity: Data Integrity ensures data does not get altered or tampered.

System availability: System should be useable and available.

This chapter discusses the theory of cryptography and user authentication which can be used to provide most of the security services described above.

In practice, there are many ways to attack an operating system. The attacks can be from inside of the system, or from outside. For the attacks from inside, topics such as Trojan horses, login spoofing, logic bombs, trap doors, buffer overflow, generic security attacks, famous security flaws, and design principles for security are discussed in this chapter. For the attack from outside, this chapter discusses virus damage scenarios, how viruses work, how viruses spread, antivirus and anti-antivirus techniques, Internet worm, mobile code, and Java security.

Finally, this chapter provides various techniques that an operating system can use to protect files and data. These include the protection domain, access control lists, and capabilities.

Adware and Spyware

One threat to operating systems not discussed in the textbook is adware and spyware, both of which invade an operating system. Adware and spyware are not malicious like viruses or worms but they are more of a nuisance. They attach themselves in the registry and provide information back to various sites on your internet browsing and buying habits.
