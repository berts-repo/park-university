# Unit 7: Operating Systems Security

Overview

Since an operating system controls the operations of a computer, it is imperative to have a strong and secure operating system. In this unit, you will study the threats that operating systems face and the methods of strengthening the operating systems.

Learning Objectives

Upon completing this unit, students will:

Compare and contrast the difference between Computer and Network Security

Classify what potential security threats exist

Examine the levels of protection

Formulate some protection mechanisms

Analyze Intrusion Detection

Classify computer viruses and explore examples of their occurrences

Differentiate Linux and Windows security mechanisms

Continuing from Unit 6, write a program to solve consumer-producer problem
