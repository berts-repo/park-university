# Unit 5: Input and Output Principles

Overview

Operating systems spend a lot of time managing I/O and with multimedia applications becoming more popular it is imperative that operating systems optimize I/O. In this unit, we will discuss I/O, including the I/O hardware and software principles, disks, clocks, character-oriented terminals, graphical user interfaces, and network terminals.

Learning Objectives

Upon completing this unit, students will:

Evaluate the differences in I/O devices

Examine the concepts and techniques in performing I/O

Assess design objectives for addressing I/O

Analyze I/O buffering approaches

Evaluate specific techniques for I/O to disk storage

Compare and contrast I/O management in Linux and Windows

Write a multi-threaded program in Linux
