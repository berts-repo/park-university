# Unit 4: File Systems

Overview:

A file system consists of file structure, directory structure, and storage management. In this unit, you will study the file system and file management in both Windows and Unix operating systems.

Learning Objectives

Upon completing this unit, students will:

Differentiate the concepts of file management

Examine the basic organizations of files

Examine the concepts of directory structures of files

Analyze use of access rights for security

Assess the concepts and techniques for file allocation

Compare and contrast file management in Linux and Windows

Write a program to solve consumer and producer problem.
