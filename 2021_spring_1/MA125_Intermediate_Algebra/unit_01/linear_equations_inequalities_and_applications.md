# Unit 1: Linear Equations, Inequalities, and Applications

Introduction

As you will see in this unit, the concept of balance is useful in solving linear equations and inequalities. In addition, you will learn about some of the applications of these types of equations to diverse fields like: geometry, athletics, healthcare, finance, chemistry, and physics.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Solve linear equations in one variable. (CLO 2)

Solve a formula for a specified variable. (CLO 1)

Solve word problems involving percentages. (CLO 5)

Solve word problems leading to linear equations in one variable. (CLO 2 and 5)

Solve linear inequalities in one variable. (CLO 2)
