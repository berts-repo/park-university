# Unit 2: Linear Equations, Graphs and Functions

Introduction

In this unit you will learn about linear equations in two variables and their straight line graphs. One of the key concepts in this unit is the steepness or slope of such lines. Finally, you will learn about function notation, which has many applications.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Graph linear equations in two variables. (CLO 11)

Find the slope of a line. (CLO 1)

Graph a line using its slope and a point on the line. (CLO 11)

Write the equation of a line given various data. (CLO 1)

Graph linear inequalities in two variables. (CLO 11)

Evaluate functions. (CLO 1)
