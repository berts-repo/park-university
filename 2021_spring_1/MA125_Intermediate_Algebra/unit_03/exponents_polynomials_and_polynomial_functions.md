# Unit 3: Exponents, Polynomials, and Polynomial Functions

Introduction

In this unit you will learn the exponent rules, polynomial arithmetic, and some properties of polynomial graphs. In addition, you will learn about scientific notation, which can be used to express large numbers, like the size of the known universe, and small numbers, like the size of an atom, efficiently.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Simplify expressions using the exponent rules. (CLO 3)

Convert between scientific notation and standard notation. (CLO 1)

Add and subtract polynomials. (CLO 4)

Multiply polynomials. (CLO 4)

Divide polynomials. (CLO 4)

Compose polynomial functions. (CLO 1)
