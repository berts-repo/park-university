While your answer here is correct, there is an easier way to simplify this expression.

Rather than multiplying out each factor, we can use the rules of exponents.

When we multiply "like bases," we keep the base and add the exponents.

In this case, the base of (3/2) is the same.

As such, we can keep this base and add the exponents.

The sum of the exponents is:  (8) + (-5) + (-4)  =  -1

So, we have...

Since there are no "+" or "-" within the ( ), we can make the negative exponent positive by reciprocating the base.

(3/2)^-1  =  (2/3)^1  =  2/3
