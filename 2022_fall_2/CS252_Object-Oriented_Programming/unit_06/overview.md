# Unit 6: Overview

Introduction

Now that we understand inheritance, we’re ready to build Graphic User Interface (GUI) based programs! We will learn how to use common GUI components such as windows, labels, textfields, and buttons. Understanding event handling and its implementation is essential to building a GUI program.

ULO #3: Handle action events in a GUI program. (CLO 2 and 3)

ULO #4: Analyze the syntax and execution of a Java program. (CLO 3)
