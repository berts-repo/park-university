# Additional Notes on the Final Exam

You should have read the  [link: https://canvas.park.edu/courses/71018/pages/study-guide-for-cs252-final-exam-online-sessions] Final Exam Study Guide under the Course Information module.

Here are some additional notes on the final exam:

Those items won't be on the test:

which class to import

file I/O

output formatting: System.out.printf(), String.format()

source code of a Java API class, such as ArrayList, Object.

You're expected to be able to build and work with a class with array/ArrayList data members

Exception handling

relationships between Throwable, Error, Exception and RuntimeException classes. Meaning of checked vs. unchecked. IOException is the only checked exception that will be tested i.e. you may assume all other XxxException classes are descendants of RuntimeException.

 You're expected to know common exceptions such as the ones related to null pointer, invalid string/array/ArrayList index, input type mismatch, number format exception (Integer.parseInt() and similar), divide by zero, invalid type cast.  

If a GUI coding question shows up, you will be given a partial program with at least the main window set up and most GUI components ready. You won't be asked to build a complete GUI program.
