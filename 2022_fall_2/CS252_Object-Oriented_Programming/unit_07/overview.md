# Unit 7: Overview

Introduction

This unit will explain exceptions, those run-time errors you may have seen when working on your projects. The types of exceptions we usually see are caused by programmer oversight, bad user input or file input/output problems. We will learn how to handle different types of exceptions and practice in both text-based interface programs and GUI programs. We will also see GUI programs involving external data.

Click here for Unit Learning Outcomes

ULO #1: Distinguish checked and unchecked exceptions. (CLO 3)

ULO #2: Handle checked or unchecked exceptions in a program. (CLO 2)

ULO #3: Build a GUI program involving external data. (CLO 3)
