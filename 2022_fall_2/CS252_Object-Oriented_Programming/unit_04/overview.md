# Unit 4: Overview

Introduction

This unit covers how to define a class of our own. A programmer-defined class allows us to bundle a grouping of data (which becomes data members of a class) and operations that can be performed on that data (which becomes method members of a class). This is called encapsulation, the #1 principle of OOP. This unit will cover how to define a class, including declaring data members, method members, and constructors.
