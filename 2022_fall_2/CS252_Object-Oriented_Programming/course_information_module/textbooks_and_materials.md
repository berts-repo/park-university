Software

You need to install a Java Development Kit a.k.a. JDK (if you don't have one) and then an appropriate IDE to develop Java programs on your own computer. When we get to GUI programming later in this course, you also need to set up JavaFX with your IDE. See tutorials in  [link: https://canvas.park.edu/courses/71018/pages/software-faq] Software FAQ. Here is a brief explanation of what's needed:

(JDK) The latest Long-Term Support (LTS) version of an OpenJDK is preferred (currently LTS 17), such as  [link: https://adoptium.net/] Eclipse Adoptium (successor to AdoptOpenJDK). See online resources such as  [link: https://docs.google.com/document/d/1nFGazvrCvHMZJgFstlbzoHjpAVwv5DEdnaBr_5pKuHo/] Java Is Still Free  for an explanation of JDK vs. OpenJDK and versions.

(JavaFX for GUI programming) JavaFX must be downloaded separately from the OpenJDK/JDK. Download JavaFX at  [link: https://gluonhq.com/products/javafx/] Java FX- Gluon right after you download JDK as they need to be in the same version (just the main version numbers, like JavaFX 17.y.yy for JDK 17.x.xx or JavaFX 18.y.yy for JDK 18.x.xx). You need to configure JavaFX with your IDE.

Integrated Development Environment (IDE): we should learn to use an industry IDE such as  [link: http://www.eclipse.org/downloads/] Eclipse. IDE related instructions in this course are based on Eclipse.

Other possible IDEs like  [link: https://netbeans.org/downloads/] NetBeans work too.  [link: https://code.visualstudio.com/docs/languages/java] Visual Studio Code (VS Code) is a text editor (even though it's very flexible) and requires extensions and additional configurations for Java Development and JavaFX. You will need to research how to do certain things in those IDE/editors.   

Our zybooks text provides an embedded programming environment that's okay for learning, but you should learn to use a real IDE. 

You may use  [link: https://parkuniversity.freshdesk.com/support/solutions/articles/6000177801-how-do-i-access-the-computer-lab-applications-from-a-personal-computer-device-] Park's Virtual Lab before getting your computer ready. Choose "CS_Lab" from the Desktop tab. The virtual desktop has AdoptOpenJDK, JavaFX, and Eclipse (IDE) ready. You just need to configure JavaFX with Eclipse before building GUI programs.

An online Java compiler (with IDE interface) such as  [link: https://www.onlinegdb.com/online_java_compiler] Online Java Compiler may be helpful when you need to test snippets of code or when your own IDE/JDK is temporarily unavailable. Such environments may not support GUI programming though.
