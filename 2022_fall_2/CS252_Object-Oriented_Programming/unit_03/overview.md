# Unit 3: Overview

Introduction

Searching and sorting are common operations on lists of items. This unit will cover linear search, binary search, selection sort and insertion sort. We will also discuss the time efficiency of different algorithms using the Big O notation.

The arrays we introduced in the last unit are referred to as one dimensional arrays. Arrays of higher dimensions are used in software too. For example, two-dimensional arrays are commonly used to implement matrix and 2D images. We will learn the basics of two-dimensional arrays and their processing in this unit. The same principles apply to higher-dimensional arrays.
