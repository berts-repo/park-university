Required Software:

Operating Systems

This course requires students to use a Windows, Mac OS X, Linux computer or any Chromebooks or tablet. An administrative permission is not required but recommended on computers. A Windows system is preferred than Mac OS X or Linux. Students who use a Chromebooks or Tablet needs to use Citrix Virtual Desktop Environment to perform the Lab.

GENI Connection

This course requires students to set up lab environment on  [link: https://www.geni.net/] GENI system, a free open academic cloud. Students needs to deploy the lab environment through web browser and connect the lab environment with SSH client and VNC viewer. On a Windows system, a Putty is the expected SSH client. On a Mac/Linux system, the system integrated ssh command is the expected SSH client. The detailed instruction will be provided in the corresponding labs.
