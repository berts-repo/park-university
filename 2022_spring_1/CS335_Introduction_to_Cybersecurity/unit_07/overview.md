# Unit 7: Overview

Introduction

In this unit, you will touch many security concepts, including different security policies, models and trust management, different access-control models, security standards, and software vulnerabilities, different security administration and auditing procedures, as well as Kerberos authentication system. You need to complete the required tasks in the below Readings and Materials section, as well as the Assignments section. You are encouraged to complete the optional and recommended tasks.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Explain the concept of security policy, security models and trust management. (CLO 2)

Compare and classify different access control models. (CLO 2)

Explain security standards and evaluation. (CLO 2)

Explain various software vulnerability assessment methods (CLO 2)

Explain basic principles of system administration and penetration testing (CLO 2)

Explain the principles of Kerberos authentication. (CLO 2)

Explain different secure storage techniques. (CLO 2)
