# Unit 5: Overview

Introduction

In this unit, you will explore the basics of the web system, i.e., HTTP and HTML. It will facilitate you to better understand web attacks and countermeasures, which will be introduced in next chapter. We will also study HTTPS(hypertext transfer protocol over secure socket layer), the secured version of HTTP protected by digital certificate. In this week, we will perform our first web security lab, learning how to intercepting and changing HTTP request and response with OWASP ZAP. You need to complete the required tasks in the below Readings and Materials section, as well as the Assignments section. You are encouraged to complete the optional and recommended tasks.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Explain and analyze the HTTP request and response packet (CLO 2,3)

Explain and analyze the HTML dynamic content and HTTP sessions and cookies (CLO 2,3)

Explain and analyze HTML document. (CLO 2,3)

Discuss and identify various Web-based attacks (CLO 3)

Experiment with intercepting and modifying HTTP request and response (CLO 4)

Explain the design and technical details about HTTPS and Web Server Certificate (CLO 4)
