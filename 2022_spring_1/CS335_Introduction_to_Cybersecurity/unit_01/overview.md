# Unit 1: Overview

Introduction

In this unit, you will be introduced to the basic concepts of cybersecurity. Also, you will learn the basic commands of the Linux system. You will also be required to complete Lab 1, which will guide you to set up the vPark lab environment that will be used in the following units. You need to complete the required tasks in the below Readings and Materials section, as well as the Assignments section. You are encouraged to complete the optional and recommended tasks.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Describe the challenge of computer security (CLO 3)

Identify and categorize various types of threats and attackers (CLO 3)

List and explain security principles. (CLO 2)

Use basic Linux commands (CLO 4)

Explain and operate on Linux file system and files (CLO 2, 4)

Explain and use administrative access on Linux (CLO 4)
