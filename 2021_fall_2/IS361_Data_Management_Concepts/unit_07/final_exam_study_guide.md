# Final Exam Study Guide

The Final Exam for this course comprises most of the topics covered in the past weeks. In particular the students should be prepared for the following tasks:

Understand and apply basic concepts on databases: Data, Information, Metadata, File processing, Database Management Systems, Entities, Attributes, Relationships, Identifying Attributes, Multivalued Attributes, Composite Attributes, Tables, Rows, Columns, Files, Records, Fields, Primary Keys, Foreign Keys, Database Partitions, File Organizations, Two and three tier approaches

Understand and be able to identify violations of Integrity Constraints in databases.

Understand relational data models and how to obtain them from ERDs. Be able to answer questions about a drawn relational database model.

Understand normalization and be able to normalize a un-normalized table to 1NF, 2NF and 3NF.

Be able to write correct SQL SELECT queries to retrieve information from a relational database model.

Be able to design a Data-Mart using the star-schema from specifications and a relational database model.

Be able to differentiate Data Administrator functions from Database Administrator functions.

The Final Exam will include True/False, Multiple Choice, and short answers for a total of 100 points.

The final exam for this course has the following characteristics:

It is proctored.

It is a 2-hour final that can only be taken once. (This is school and department policy)

It is closed books, closed notes

No use of calculators is allowed.

No use of websites allowed, other than the exam itself.

Multiples sheets of scratch paper are allowed.

No use of additional software is allowed

Bathroom breaks are not allowed. Students should take care of their needs before starting the exam.

Only the Chrome Browser is allowed.

Each question in the exam will be presented to the student individually and students will not be able to go back to a question after it was answered and submitted

Additional Examinations Notes:
Students may be flagged and warned about infractions while taking the exam, but they should be allowed to complete it within its allocated time period.
