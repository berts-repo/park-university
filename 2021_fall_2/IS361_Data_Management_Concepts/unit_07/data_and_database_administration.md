# Unit 7: Data and Database Administration

Introduction

This unit explores the roles of data administrators and database administrators. It explains the differences between the two roles, and describe their most important tasks. These include: data and database security, following of data standards, integrity controls, backup and recovery, granting access to data, concurrency, and performance tuning, among others.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Compare the roles of Data Administrator and Database Administrator (CLO1)

Distinguish the main administrative tasks in database administration (CLO 1)

Appraise the quality of data in a database system for integration in a data warehouse (CLO5).
