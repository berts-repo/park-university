Course Details

Title:  IS361 Data Management Concepts

Course Description: This course provides an overview of data management concepts. This course explores the enterprise perspective of managing data needs of an organization. This includes data integrity, database models, and integration of databases, security, and database administration issues. The student will be introduced to query processing within a database environment.

Credit Hours: 3:0:3

Prerequisite Courses: CS208 and either CS151 or CS152.

Core Learning Outcomes

At the conclusion of the course, learners will be able to:

Distinguish the main components, organization, and administration of database systems.

Create an Entity Relationship Diagram (ERD) from a set of written requirements.

Create a Relational database from an Entity Relationship (ER) Diagram.

Construct Structured Query Language (SQL) commands for database queries and database administration.

Compare the design and operation of data warehouses with transactional databases.
