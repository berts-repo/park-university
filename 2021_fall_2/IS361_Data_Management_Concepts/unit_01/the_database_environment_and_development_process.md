# Unit 1: The Database Environment and Development Process

Introduction

In this unit, we will become familiar with the basic concepts and elements of a database environment and the process for the development of a database system.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Distinguish the main components of database systems (CLO 1).

Compare file processing systems against database management systems (CLO 1).

Distinguish the main stages of the development of database management systems (CLO 1).
