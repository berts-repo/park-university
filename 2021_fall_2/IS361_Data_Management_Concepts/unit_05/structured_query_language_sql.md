# Unit 5: Structured Query Language (SQL)

Introduction

Unit 5 presents Structured Query Language (SQL). This is the most widely used language for creation, manipulation and query of relational databases. Students will gain hands-on experience retrieving information from databases using SQL query commands. The basic SELECT command will be explained, and further enhanced with the use of wild cards, aggregators and subqueries.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Construct Structured Query Language (SQL) commands for creation and updating of relational database tables (CLO 4).

Construct Structured Query Language (SQL) commands for database queries (CLO 4).
