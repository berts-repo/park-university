# Unit 5: Content Highlights - Simple Query Language (SQL)

This page highlights

the main points on SQL, and complement some of the textbook explanations. It is recommended to first read this week’s assigned chapters from the textbook, and then come back to this document for further commentary.

The highlights of this week’s readings are:

The most important chapter this week is Chapter 6, that contains the basics of SQL commands, and in particular, the basics of the SELECT command used for queries. Chapter 7 goes deeper into the philosophy and capabilities of SQL commands. It is a good read if you are up to it, but it is optional at this stage. For this course we will be satisfied with students knowing the commands of Chapter 6.

As indicated above, the SELECT command is the most important command for database queries, therefore students should become familiar with it. Please review the additional document prepared for this week that shows examples of the many uses this command has. Not only read about SQL, but practice along with these exercises in the database provided for Assignment 5. When you feel confident, tackle the assignment in a fresh copy of the file. Use also the discussion thread to practice more and discuss in class on SQL commands. The more you practice, the more you will understand how SQL works.

Even though the SELCT command takes the lion’s share this week. There are other SQL commands that help you create, modify and delete tables, records and fields. Microsoft Access provides an easier environment to perform all these activities, so the emphasis is not in these commands. Review their format in the textbook and try to use them in the discussion thread.
