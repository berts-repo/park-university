# Unit 8: Big Data and Analytics

Introduction

In this final unit, we briefly explore the concept of Big Data, or the manipulation of enormous amount of data using alternative techniques to the ones used in relational databases. We review trends in this new area of development and some of the current products being used in this area.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Distinguish the main issues with Big Data (CLO 1).
