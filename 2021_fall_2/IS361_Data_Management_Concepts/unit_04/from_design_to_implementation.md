# Unit 4: From Design to Implementation

Introduction

In this unit, students will convert a relational data model into a real database using Microsoft Access.  Students will practice the creation of tables with appropriate data types for all fields and unique primary keys. They will also practice the establishment of relations between tables using foreign keys.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Create a physical database that follows the relational data model (CLO 3).

Compare two-tier and three-tier architectures for database implementations (CLO1, CLO3).
