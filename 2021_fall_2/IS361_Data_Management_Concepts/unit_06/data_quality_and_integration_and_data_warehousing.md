# Unit 6: Data Quality and Integration and Data Warehousing

Introduction

In this unit, we cover some issues related to data quality in databases and integration of data from various sources. We begin talking about data warehouses, because they will provide a natural platform for talking about data quality and data integration. We will be particularly interested in data marts, smaller versions of data warehouses, and the star schema used for the architecture of data warehouses.  The second part of this unit will discuss the importance of data quality and integration, and present the ETL process to extract, transform, and load data into transactional databases and data warehouses.

Unit Learning Outcomes

At the conclusion of the unit, the learner will be able to:

Distinguish the main components of data warehouses and data marts (CLO1, CLO5)

Compare Data Warehouses against Transactional Databases (CLO1, CLO5)

Design Entity Relationship (ER) Diagrams for Data Marts, using the star architecture (CLO5)
