# Unit 5: Overview

Introduction

Reading a Novel

In this unit, you will do the following:

Read the Overview and Lecture

Read the first half of the assigned novel

Write and submit your reading journal

Participate in the discussion (1 initial post, 2 responses)

Write and submit Short Paper 2

Start thinking about the final Core Assessment paper

Unit Learning Outcomes

At the completion of this unit, the learner will be able to:

Argue a claim about a fictional text. (CLO 3)

Support the claim with evidence from the primary text. (CLO 1, 2, and 3)

Explain how biographical or historical context can influence a fictional text. (CLO 3)
