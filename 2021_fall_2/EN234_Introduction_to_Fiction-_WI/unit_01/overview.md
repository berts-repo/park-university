# Unit 1: Overview

Introduction

What is Fiction?

In this unit, you will do the following: 

Post your introduction

Read the Overview and Lecture

Read the assigned texts

Write and submit your reading journal

Participate in the discussion (1 initial post, 2 responses)

Start thinking about the Midterm Project

Unit Learning Outcomes

At the completion of this unit, the learner will be able to:

Define the basic genre of fiction. (CLO 1)

Recognize features of fiction, which might include plot, character, setting, and/or theme. (CLO 1)

Practice how to read a fictional text. (CLO 1 and 2)

Practice discussing a fictional text in relation to the features of fiction. (CLO 2 and 3)
